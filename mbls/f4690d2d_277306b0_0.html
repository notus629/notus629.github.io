Apache 配置

    服务器级配置文件： httpd.conf (http daemon configuration)

        // 配置文件路径（服务器级的, Windows）
        Apache2.2/conf/httpd.conf


    虚拟主机配置文件： httpd-vhosts.conf (httpd virtual hosts configuration)

        Apache2.2/conf/httpd-vhosts.conf


    目录级配置文件： .htaccess （http access?）

        // 目录级的（各目录下的 .htaccess 文件）
        // 可以在命令行下使用 “echo a > .htaccess” 建立 .htaccess 文件
        // 需要将其上的 httpd.conf 和 httd-vhosts.conf 中设置 AllowOverride all
        各具体目录/.htaccess

    

上下文环境 - Context

    httpd.conf                                    per-server context

    <VirtualHost> blocks                          per-virtualhost context

    .htaccess files and <Directory> blocks        per-directory context
