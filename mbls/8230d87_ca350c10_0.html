php memcache 扩展 - 创建对象实例与连接服务器

    1. 创建一个 Memcache 对象实例

        $mem = new Memcache;


    2. 连接 memcached 服务器

        if( !$mem->connect("127.0.0.1", 11211) ){
            die("连接失败!");
        }
