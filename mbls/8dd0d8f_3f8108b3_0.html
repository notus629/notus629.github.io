环境

    系统： Windows

    Apache 2.2



步骤如下

    1. 加载 Rewrite 模块

        对于 conf/httpd.conf 文件，找到

            LoadModule rewrite_module modules/mod_rewrite.so

        默认为注释，去掉前面的注释“#”号

    
    2. 允许在各目录中使用.htaccess文件（即用.htaccess的设置对httpd.conf进行覆盖?)

        在 httpd.conf 中，将 AllowOverride 的值改为 All (默认为 None), 即

            AllowOverride All

        由于 windows 中不允许建立 .htaccess 文件，在命令行下，使用 echo a > .htaccess 来建立。
