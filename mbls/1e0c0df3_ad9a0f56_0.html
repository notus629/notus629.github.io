3 种主要连接操作

    用于 MySQL 的主要 API 有三种： mysql 扩展，mysqli 扩展，PHP 数据对象 PDO

    

MySQL 的用户配置

    $host = 'localhost';   // 连接的数据库端口号
    $user = 'root';        // 连接数据库的用户名
    $pass = '';            // 连接数据库的密码



mysql 扩展连接数据库 - （从php 5.5开始弃用， php7 移除，不推荐使用）

    // 使用 mysql_connect 函数进行数据库连接
    $mysql = mysql_connect($host, $user, $pass); 
    // 通过 mysql_select_db 函数来选择数据库
    mysql_select_db('imooc'); 



mysqli 扩展连接数据库

    // 面向对象的方法实例化 mysqli 对象
    $mysqli = new mysqli($host,$user,$pass,'imooc'); 



PDO 数据库抽象层访问数据库

    $dsn = 'mysql:host=localhost;dbname=imooc';
    // 实例化PDO对象
    $pdo = new PDO($dsn,$user,$pass);
