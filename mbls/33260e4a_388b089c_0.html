安装步骤

    1. 防止在安装过程中（如因网络掉线、putty被关）lnmp被中断，在安装 lnmp 前执行 screen 命令

        ## 创建 screen 会话
        $ screen -S lnmp


    2. 下载并安装LNMP一键安装包（可使用离线方式）

        $ wget -c http://soft.vpser.net/lnmp/lnmp1.3.tar.gz && tar zxf lnmp1.3.tar.gz && cd lnmp1.3 && ./install.sh lnmp

        中间会让选择安装的版本与安装包

        除去下载时间，整个安装过程大约需要半小时，最后提示： "install lnmp V1.3 completed! enjoy it.", 则表示安装成功。



注意：

    对于 mysql 的新版本，此安装包提示要求硬件内存大于1G，否则会安装出错。
