Date 对象

    创建 Date 对象，使用构造函数语法

        // 默认存储当前日期，存储其他日期则在参数中指定
        // JS 中日期以数字存储，即从1970年1月1日到现在为止的毫秒数
        var today = new Date();

     
    以下 3 种设置格式设置日期

        // YYYY, MM, DD, HH, MM, SS
        var dob = new Date(1996, 11, 26, 15, 45, 55);  

        // MMM DD, YYYY HH:MM:SS      
        var dob = new Date ('Dec 26, 1996 15:45:55');        
                               
        var dob = new Date(1996, 11, 26);
