对象 & 数组
       
    变量已被声明但还未赋值时，它就是 undefined

    若要通过属性名或键来访问项，使用对象（键唯一）， 属性的顺序不重要
                       
    若顺序重要，使用数组，数组是特殊类型的对象：键为索引编号



对象中的数组

    costs = { 
              room1: items[420, 40, 10],
              room2: items[460, 20, 20]  
            }

    访问： costs.room1.items[0];



数组中的对象
               
    costs = [  
              {accom: 420, food:40, phone: 10},
              {accom: 460, food:20, phone: 20} 
            ]
               
    访问：costs[1].phone;
