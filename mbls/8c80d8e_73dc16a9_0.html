SSH 实现多主机间无密登陆

    1、添加非 root 用户：$ useradd 用户名，修改密码： $ passwd 用户名

    2、通过 $ ssh-keygen -t rsa 生成密钥信息两个文件，期间可以选择俩文件的生成路径，这里选择默认地址：~/.ssh

    3、通过 $ scp /home/ct/.ssh/id_rsa.pub <用户名>@<远程主机 IP>:~/.ssh/ 将生成的公钥文件 id_rsa.pub 拷贝到远程主机的 ~/.ssh 目录下面

    4、重命名： # cat id_dsa.pub >> ~/.ssh/authorized_keys

    5、修改文件权限（这一步特别重要）
        $ chmod 600 authorized_keys
        $ chmod 700 -R .ssh

    6、在源主机上使用：$ ssh <用户名>@<远程主机ip>，进行登录测试，无需再次输入密码；

    7、如果测试不成功，重点看第5步，该部分知识在自动化部署这一块，应用比较广泛，同时也能够有效提高效率！
