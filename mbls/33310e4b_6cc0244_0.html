Pyrus [1]

    Pyrus is a tool to manage PEAR packages. Pyrus simplifies and improves the PEAR experience.

    安装：直接使用 $ php pyrus.phar 即可



基本使用

    ## Set where to install packages:
    $ php pyrus.phar mypear ~/src/my-project/vendor

    ## Install a package:
    $ php pyrus.phar install PEAR2_HTTP_Request

    ## List installed packages:
    $ php pyrus.phar list-packages



示例

    ## 安装 Mail_Mime 包
    $ php pyrus.phar install pear/Mail_Mime
