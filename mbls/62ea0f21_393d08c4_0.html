键盘事件

事件触发
input当 input 或 textarea 元素的值发生变化时触发
keydown按下键盘上的任意键时触发，按住一个键不放，就会持续触发
keypress按下一个键，并在屏幕上反映为一个字符时触发

当用户按下方向键时不会触发 keypress

按住一个键不放也会持续触发
keyup松开键盘上的一个键时触发

keydown, keypress 发生在字符显示在屏幕之前

keyup 发生在字符显示在屏幕之后


    顺序： keydown -> keypress -> keyup
