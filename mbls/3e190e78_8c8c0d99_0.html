第3章        使用数组
PHP支持：索引数组 & 关联数组。
3.1        什么是数组(p55)
标量变量 & 数组
索引（关键字）：用来访问数组中的元素。
PHP允许间隔性地使用数字或字符串作为数组的索引。

3.2        数字索引数组(p56)
数字索引默认从0开始
3.2.1 数字索引数组的初始化
  1. 利用array()(语言结构，非函数，类似echo)初始化：
$products = array( 'Tires', 'Oil', 'Spark Plugs' );
  1. 运算符"="，将数组复制到另一个数组
  1. 利用range()函数创建升序排列的数字/字符数组：
$numbers = range(1, 10);
$odds = range(1, 10, 2);        // step = 2，can be negative
$letters = range('a', 'z');
  1. 可以直接从文件或数据库中直接载入数组
3.2.2 访问数组的内容(p57)
  1. 访问数组：$products[0]
  2. 改变数组内容：$products[1] = 'Fuses';
  1. 增加一个元素到数组末尾：$products[3] = 'Uber'; //原数组大小为3
数组不需要预先初始化或创建，第一次使用时自动创建，即
$products[0] = 'Tires';
$products[1] = 'Oil';
$products[2] = 'Spark Plugs';
3.2.3        使用循环访问数组(p57)
for循环：
for ($i = 0; $i < 3; $i++){
       echo $products[$i]." "; }
foreach循环：(p58)
foreach ($products as $current){
               echo $current." "; }

3.3        使用不同索引的数组
索引数组：每个元素一个默认的索引（0、1等）。
关联数组：将变量值与任何关键字或索引关联起来。
3.3.1        初始化关联数组(p58)
$prices = array('Tires'=>100, 'Oil'=>10, 'Spark Plugs'=>4);
关键字和值之间：在大于符号前面加上等于符号，即=>
3.3.2        访问数组元素
访问数组
$prices['Tires']
创建并添加元素：
$prices = array('Tires'=>100);
$prices['Oil'] = 10;
$prices['Spark Plugs'] = 4;
添加第一个元素的同时创建数组：
$prices['Tires'] = 100;
$prices['Oil'] = 10;
$prices['Spark Plugs'] = 4;
3.3.3        使用循环语句(p58)
可以使用foreach循环或list()或each()结构，无法使用for循环。
foreach($prices as $key=>$value){
               echo $key." = ".$value."<br />;
}
each()结构：(p59)
       while ($element = each($prices) {
           echo $element['key'];
               echo "-";
               echo $element['value'];
               echo "<br />";
       }
*        each()返回数组的当前元素，并将下一个元素作为当前元素。
*        位置key和0包含了当前元素的关键字，而位置value和1包含了当前元素的值。
list()函数，将一个数组分解为一系列的值。
while( list($product, $price) = each($prices) ) {
               echo "$product - $price<br />";
}
reset()函数，将当前元素重新设置到数组开始处。（使用each后当前元素位置会变化。）

3.4        数组操作符(p60)
  1. +， 联合，如$a + $b，$b将被附加到$a中。$b与$a中索引相同(关键字冲突)的元素不添加。
  2. ==，等价，$a == $b, 包含相同元素则返回true
  3. ===, 恒等， 包含相同的顺序和类型的元素
  4. !=, <>, !==

3.5        多维数组
定义
$products = array( array ( 'TIR', 'Tires', 100),
                         array ( 'OIL', 'Oil', 10),
                         array ( 'SPK', 'Spark Plugs', 4) );
访问：
*        products[0][0]
*        还可利用双重for循环
使用列名称来代替数字索引：(即行数字索引，列是关联索引)
$products = array ( array ( 'Code'=>'TIR',
                                       Description'=>'Oil',
                                       'Price'=>10          ),
                                               ……
                         );
访问方式：行数字索引+列名称索引
       $products[$row]['Code']
三维数组：高、宽、深；
php数组没有维数限制。

3.6        数组排序
3.6.1        使用sort()函数
对数组products进行按字母/数字升序排序：sort($products);
sort()函数
*        区分字母大小写，大写字母在小写字母前面
*        默认按字母、数字升序排序
*        函数第二个参数可选，可为SORT_REGULAR(默认值), SORT_NUMERIC, SORT_STRING
3.6.2        使用asort()函数和ksort()函数对关联数组排序
关键字和值在排序时仍然保持一致
*        asort()，根据数组的每一个元素值进行排序。
*        ksort()，根据数组的key即关键字进行排序
3.6.3        反向排序
rsort(), arsort(), krsort()，与sort(), assort(), ksort()相反，将数组降序排序。

3.7        多维数组的排序（p64）
函数usort()，告诉PHP如何比较各个元素。默认按照compare所定义的升序来排列。
usort($products, 'compare');
function compare($x, $y){
       if ($x[1] == $y[1]) {
               return 0;
       } else if ($x[1] < $y[1]) {
               return -1;
       } else {
               return 1;
       }
}
对应的uasort(), uksort()，要求传入用户定义的比较函数。

3.8        对数组进行重新排序（p66）
shuffle()函数，将数组各元素进行随机排序。修改原数组。
array_reverse()函数，给出一个原来数组的反向排序。返回原数组修改后的副本，可直接用来覆盖原数组。
array_push()， 将每个新元素添加到数组的末尾。
array_pop()，删除并返回数组末尾的一个元素。

3.9        从文件载入数组(p68)
count()函数，统计数组中元素的个数，利用file()函数载入文件到数组。
explode()函数，将传入的字符串分割成一个个小块。
array explode(string separator, string string [, int limit])
*        参数limit,限制被返回的最大块数
intval()函数，将一个字符串转化成整数，即从中提取数字，忽略不能转化的部分。

3.10        执行其它的数组操作
3.10.1 在数组中浏览(p71)
数组内部指针：指向数组中的当前元素。
current($array_name)，指向当前元素，同pos()
each($array_name)，在指针前移一个位置之前返回当前元素。
next($array_name)，将指针前移，然后再返回新的当前元素。
reset($array_name)，返回指向数组第一个元素的指针，指针移到开头。
end($array_name)，返回数组的最后一个元素，指针移到末尾。
prev($array_name)，将当前指针往回移一个位置，然后再返回新的当前元素。
pos($array_name), Alias of current

3.10.2 对数组每个元素应用函数(p72)
array_walk()，以相同方式使用或者修改数组中的每一个元素。
bool array_walk(array arr, string func, [mixed userdata])
*        func，用户自定义的作用于每个元素的函数
*        userdata，可以作为一个参数传递给自己的函数
yourfunction(value, key, userdata)，用户自定义的函数，可以忽略关键字参数和userdata参数。

3.10.3 统计数组元素个数
count()，对数组中元素的个数进行统计
sizeof()，同count()
array_count_values()
*        统计每个特定的值在数组中出现过的次数（数组的基数值）。
*        返回一个包含频率表的关联数组。（以值作为关联数组的关键字）。

3.10.4 将数组转换成标量变量(P74)
extract()，创建数组对应的标量变量，针对非数字索引数组
extract(array var_array [, int extract_type] [, string prefix] );
*        标量变量的名称必须是数组中关键字的名称，值则是数组中的值。
*        extract_type, 如何处理名字冲突，默认覆盖已有变量(p74，表3-2)
*        prefix, 冲突时可能需要的参数
