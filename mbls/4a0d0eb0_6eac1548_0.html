全局安装(推荐)

    先下载 composer.phar 文件，composer 的二进制归档文件，让用户在命令行下执行一些操作


    将 Composer 安装到系统环境变量 path 所包含的路径下面，即可在命令行窗口中直接执行 composer 命令, 不需要在前面加上 php.


    Mac 或 Linux 系统
        ## 拷贝/移动+改名
        sudo mv composer.phar /usr/local/bin/composer 
        ## 加上执行权限
        sudo chmod -R 755 composer


    Windows 系统

        将 composer.phar 拷贝到 php.exe 同级目录，将建composer.bat文件，并将下面代码保存到该文件中：

        @php "%~dp0composer.phar" %*



composer.phar 下载方式

    方法一： 直接下载


    方法二： 通过命令行(须先设置好 php 环境变量，用 php -v 检验) 

        php -r "readfile('https://getcomposer.org/installer');" | php
