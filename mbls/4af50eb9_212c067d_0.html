真值和假值

    因为存在强制类型转换， js 中的每个值都可被当作 true 或 false 处理

    null 和 undefined 除了自身之外不等于任何值

    NaN 不等于任何值，也不等于它自己
  
  

假值

    false

    0

    ''           空串，而非空格： ' '

    NaN          如 10/'score'

    undefined    如： var highScore;

    null



真值

    其它情况

        true, 1, 'carrot', 10/5, 'true', '0', 'false' ...

        对象或数组通常会被认为真值
