    2. 如何将 Session 数据放入到 memcached

        默认是将 Session 放入到文件

        a. 修改 php.ini

            session.save_handler = files | user | memcache

            files:  默认值，放入到某个路径（由 session.save_path 设置
            user: 自定义
            memcache

            设置如下

                session.save_handler = memcache
                session.save_path = "tcp://127.0.0.1:11211"

<?php
 
    // 传统的代码将 session 保存到文件
    // php.ini 配置后可以保存到 memcached
 
     session_start();
     $_SESSION['name'] = '天龙八部';
     $_SESSION['city'] = 'beijing';
 
     class Dog{
       public $name;
       public function __construct($name){
           $this->name = $name;
       }
     }
 
     $dog1 = new Dog('xinxin');
     $_SESSION['dog'] = $dog1;      // 将 session 中的对象保存到 memcached
 
 
 
     // 如果 session 数据入 memcached, 那他一定是以 session_id 为 key 值进行添加
     // 取出
     $name = $_SESSION['name'];
     echo "name=$name";
     echo "sessionid=".session_id();


        c. 若管理员不允许修改 php.ini 文件，可在 php 代码中利用 ini_set()函数，处理 session 入 memcached 的配置

            ini_set("session.save_handler", "memcache");
            ini_set("session.save_path", "tcp://127.0.0.1:9999");

            // 同时你也可以通过 ini_set 去动态的修改对 php.ini 的其它设置
