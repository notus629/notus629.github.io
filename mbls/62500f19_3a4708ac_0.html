选择单个元素

    getElementById(...)         // 根据 ID

    querySelector(...)          // 使用 CSS 选择器，返回第一个匹配的元素



选择多个元素

    getElementsByClassName(...)        // class 属性中使用了特定值的元素

    getElementsByTagName(...)          // 使用了指定标记的元素（即元素标签）

    querySelectorAll(...)              // 使用 CSS 选择器来选择所有匹配的元素



在元素节点间遍历（以下皆为属性而非方法）

    parentNode

    previousSibling/nextSibling

    firstChild/lastChild
