重难点：

1. __set() and __get() method:
       __set() is run when writing data to inaccessible properties.
       __get() is utilized for reading data from inaccessible properties.
         __set()和__get()仅针对不可访问（即private）的属性才起作用，但这些属性在定义了__set()和__get()之后，变的在外部可以访问（读和设置）。


2.  类中属性访问时的$符号.
       访问时直接$classname->var即可。
       定义时则为$var
        在内部判断其名字时，也是与"var"比较

3.  $_SERVER['PHP_SELF'] 与 $_SERVER['HTTP_HOST']
       $_SERVER['PHP_SELF'] - 表示当前 php 文件相对于网站根目录的位置地址，如地址http://www.5idev.com/php/，可得到/php/index.php
       $_SERVER['HTTP_HOST']-- 主机域名？
               $url = "http://".$_SERVER['HTTP_HOST'].$_SERVER['PHP_SELF']; 得到：http://www.5idev.com/php/index.php

4. 花括号的作用
       一、不管什么程序，function name(){}, for(){}, ….太多了，不说也知道做什么用了。
       二、$str{4}在字符串的变量的后面跟上{}刚大括号和中括号一样都是把某个字符串变量当成数组处理。
       三、{$val}这时候大括号起的作用就是，告诉PHP，括起来的要当成变量处理。


5. PHP换行时机
        1. 字符串里面可以换行
        2. 字符连接符处可以换行
