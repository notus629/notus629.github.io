Linux 上安装 PHP 扩展

    通常命名为：extname.so


    两种安装方式

        编译方式

        手动方式



编译方式安装扩展

    Using pecl command [1]
    
        ## This will download the source for extname, compile, and 
        ## install extname.so into your extension_dir. 
        ## extname.so may then be loaded via php.ini
        $ pecl install extname


    Using phpize [2]

        ## The phpize command is used to prepare the build environment 
        ## for a PHP extension.
        $ cd extname
        $ phpize
        $ ./configure
        $ make
        $ sudo make install


    成功编译后通常会将 PHP 扩展直接放到对应的 PHP 扩展目录，只需要在 php.ini 文件中载入对应的扩展即可, 即

        extension=extname.so
