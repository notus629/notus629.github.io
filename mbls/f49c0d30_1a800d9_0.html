FAQ



1. Apache 的配置文件位置。
    
A. 位于 Apache 安装目录下的 conf/httpd.conf



2. 安装 Apache 出现，"以一种访问权限不允许的方式做了一个访问套接字的尝试"？ [1]

A.  错误背景：安装win10更新后出现。

    错误原因：端口冲突，TCP80端口已被占用。

    解决方案1：将Apache 的配置文件  Apache安装目录下的conf/httpd.conf，Listen 80，即apache的监听端口改为其它值（如88）即可。

    解决方案2：即停止占用port 80的服务，经netstat -ano查询得知占用此端口的PID为4，进一步，为NT Kernel & System服务，google之，可能使用此服务的有：

        World Wide Web Publishing Service
        Web Deployment Agent Service
        SQL Server Reporting services（最终此服务终止后解决问题，可能此服务即由win10更新后开启）
        


3. Apache 服务器打开目录后不会自动调用 index.php ?

A. 找到 apache 配置文件 httpd.conf, 然后找到并设置：
        DirectoryIndex index.php index.html
