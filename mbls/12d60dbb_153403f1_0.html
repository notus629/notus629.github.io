git remote

    ## 添加远程仓库
    ## 将 git@github.com:obpxnfxm/phppractice.git 设为本地仓库的远程仓库,
    ## 其标识符为：git-phppractice, 一般习惯命名为 origin
    $ git remote add git-phppractice git@github.com:obpxnfxm/phppractice.git


    ## 验证新的远程 URL 
    $ git remote -v


    ## show local repository related url
    ## determine the URL that a local Git repository was originally cloned from?
    $ git remote show origin
    ## or
    $ git config --get remote.origin.url
