ip

    ## 用ip命令显示网络设备的运行状态
    # ip link list

    ## -s 查看详细信息
    # ip -s link list



CentOS 7 改为动态获取 IP 地址 （即 dhcp 为 yes)

    ## 1. 查看网卡 IP 信息，找到未开启的网卡名，如 enp0s3
    ## 等同于 # ip addr
    # ip add

    ## 2. 找到网卡对应的配置文件（enp0s3对应ifcfg-enp0s3），打开进行编辑
    $ sudo vi /etc/sysconfig/network-scripts/ifcfg-enp0s3
  
    ## 3. BOOTPROTO=dhcp 表示使用 dhcp
          ONBOOT=no 默认值为 no，改为 yes, 保存后重启



CentOS/RHEL 7  Restart/Stop/Start Networking Command

    use network.service

    
    ## Get status of network service
    # systemctl status network             // or
    # systemctl status network.service    


    ## restart network service
    # systemctl restart network            // or
    # systemctl restart network.service    


    ## start network service
    # systemctl start network              // or
    # systemctl start network.service    

    ## stop network service
    # systemctl stop network               // or
    # systemctl stop network.service
