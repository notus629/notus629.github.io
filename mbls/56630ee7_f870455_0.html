函数

    匿名函数

        没有名字，不能被调用，在解释器经过时会立即执行
       

    函数的声明、调用
               
        函数可以在被声明前调用

       
    return 关键字
               
        返回值
               
        返回多个值，以数组形式返回，如
            var area = 10;
            var volume = 100;
            sizes = [area, volume];
            return sizes;

        取多个返回值：
            someFunc(10,10)[0];
            someFunc(10,10)[1];
