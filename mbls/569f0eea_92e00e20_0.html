HTML框模型的尺寸 p338


    获取和设置


    .height()           框的高度（不含内外边距和边框）

    .width()            框的宽度（不含内外边距和边框）


    .innerHeight()      框的高度 + 内边距

    .innerWidth()       框的宽度 + 内边距

 
    .outerHeight()      框的高度 + 内边距 + 边框

    .outerWidth()       框的宽度 + 内边距 + 边框


    .outerHeight(true)  框的高度 + 内边距 + 边框 + 外边距

    .outerWidth(true)   框的宽度 + 内边距 + 边框 + 外边距



注：

    1. 若单位为 px, 则单位和引号均可省略

    2. 否则，均不可省略

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>修改尺寸</title>
    <script type="text/javascript" src="js/jquery.js"></script>
    <script type="text/javascript">
        $(function(){
            var listHeight = $('#container').height();
            $('ul').after('<p>List Height: ' + listHeight +'px</p>');
            $('li').width('50%') ;
            // if the unit is px, the unit and quotes could be omitted, otherwise, both are necessary.
            $('li:eq(0)').width(125);
            $('li:eq(1)').width('75%');
        });
    </script>
    <style type="text/css">
        .hot {
            background-color: #ccc;
        }
    </style>
</head>
<body>
<div id="container">
    <ul>
        <li class="hot">PHP</li>
        <li class="hot">JavaScript</li>
        <li class="hot">Java</li>
    </ul>
</div>
</body>
</html>
