PHP 与 Web 服务器的交互方式

    主要有 SAPI, CGI, fastCGI ( 还有 Windows 的 SAPI，称作 ISAPI)

        SAPI 即是 PHP 以模块的方式与 Web 服务器交互

        CGI，以多

        fast CGI


    查看 PHP 与当前 Web Server 的交互方式

        可以使用 php_sapi_name() 函数。

        可以直接在 phpinfo 里查看 (Server API 项)
            如，在 Nginx 里： FPM/FastCGI, 在 Apache 里：Apache 2.0 Handler（即SAPI方式）
