添加/移除HTML内容

    使用 innerHTML 属性

        移除即设置为空串： ""

        有安全风险

        在 HTML 代码中使用 HTML 属性时需要转义：
            firstItem.innerHTML = '<a href=\"http://example.org\">' + ... + </a>;


    使用 DOM 操作

        利用 createElement, createTextNode, appendChild， removeChild, insertBefore

        通过DOM操作移除元素：找到元素，找到元素的父元素（parentNode），利用removeChild()移除

        速度更慢


    document.write() 方法 （不推荐）

        只在页面初始化加载时有效

        若加载完后再使用该方法：

            整个页面都会被覆盖，不是像页面添加内容，创建一个新页面



将元素添加到 DOM 树中的 4 个步骤

    a. 创建元素节点 createElement

    b. 创建文本节点 createTextNode

    c. 把文本节点添加到元素节点 appendChild / insertBefore

    d. 把元素添加到 DOM 树中
