apt-get 安装 apache2 各默认路径

    /var/www/html/index.html                    默认页面

    /usr/share/doc/apache2/README.Debian.gz     configuration system full documentation 


    /etc/apache2/                               
    /etc/apache2/apache2.conf                   file, main configuration file
    /etc/apache2/ports.conf                     file, listening ports fro incoming connections

    /etc/apache2/mods-enabled/                  directory, manage modules
    /etc/apache2/conf-enabled/                  directory, global configuration fragments
    /etc/apache2/sites-enabled/                 directory, virtual host configurations

    /etc/apache2/mods-available/                mods-enabled 通过符号连接此目录中的配置文件来激活
    /etc/apache2/conf-available/                conf-enabled 通过符号连接此目录中的配置文件来激活
    /etc/apache2/sites-available/               sites-enabled 通过符号连接此目录中的配置文件来激活


    /etc/init.d/apache2 or apache2ctl

        二进制文件（binary）名为 apache2, 在缺省设置里，用此件来进行服务的启动和终止


    /var/www/html

        Ubuntu 缺省的 Document Root

        缺省不允许访问除 /var/www, public_html directories( when enabled) and /usr/share(for web applications) 外的其它地方的文件。

        若 document root 在其它地方，需要在 /etc/apache2/apache2.conf 里头设置 whitelist
