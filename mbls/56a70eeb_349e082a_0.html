this 关键字
       
    指向当前函数所在的对象
       

    若函数在全局作用域，则 this 指代 window 对象
               
        this.innerWidth, this.innerHeight, 即 window 的属性


    所有全局变量，也都成为 window 的属性，因而在全局函数中可用 window 或 this 来访问
               
        // this.width or window.width
        var width = 500;
    
   
    在对象内定义时，this 引用的是包含此方法的对象
       

    若函数定义于全局作用域，但马上被作为对象的方法使用，则 this 代表着包含它的对象
