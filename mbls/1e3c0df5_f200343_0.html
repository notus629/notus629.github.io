PHPMailer/PHPMailer

    GitHub 上 stars 最多的邮件发送系统, 需要与第三方 SMTP 服务方通信（如QQ邮箱）来完成邮件发送

    A full-featured email creation and transfer class for PHP

    PHPMailer's integrated SMTP implementation allows email sending on Windows platforms without a local mail server.

    安装：利用 composer 进行安装，参考 [1]，在 composer.json 中加上：

        "phpmailer/phpmailer": "~5.2"

        ## 或
        $ composer require phpmailer/phpmailer
