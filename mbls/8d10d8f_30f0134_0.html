SOAP

    SOAP（Simple Object Access Protocol ） 简单对象访问协议是在分散或分布式的环境中交换信息的简单的协议，是一个基于 XML 的协议.


    它包括四个部分

        SOAP 封装(envelop)，封装定义了一个描述消息中的内容是什么，是谁发送的，谁应当接受并处理它以及如何处理它们的框架；

        SOAP 编码规则（encoding rules），用于表示应用程序需要使用的数据类型的实例; 

        SOAP RPC表示(RPC representation)，表示远程过程调用和应答的协定;

        SOAP 绑定（binding），使用底层协议交换信息。
