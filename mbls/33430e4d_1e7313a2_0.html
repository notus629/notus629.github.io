用 GitHub 来展示前端页面

    GitHub repo 上根据 master 分支 fork 一个名为 gh-pages 的分支。

    将展示页放到当前项目的 gh-pages 分支即可。

    最终页面地址为： http://(user_name|org_name).github.io/repo_name
