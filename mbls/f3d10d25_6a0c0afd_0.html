安装 python-pip 和 python-m2crypto

    ## Ubuntu 系列
    $ apt-get update
    $ apt-get upgrade
    $ apt-get install python-pip
    $ apt-get install python-m2crypto

    ## Red Hat 系(centos 64bit)
    $ yum update
    $ yum upgrade
    $ yum install python-m2ext.x86_64



查看 python 版本

    $ python --version
