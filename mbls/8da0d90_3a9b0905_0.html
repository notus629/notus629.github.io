时区设置

    默认为 GWT(UTC), 即 date.timezone = 或 date.timezone = "UTC"

    设为： date.timezone = "Asia/Shanghai"（东八区时间）

    此时，echo date(Y-m-d, H:i:s); 的输出会由标准时间变为东八区时间（php5.6.2)



date_default_timezone_set() 

    sets the default timezone used by all date/time functions.



注

    1. time() is always timezone independent. [3]

        而 date 函数对 time 进行格式化后，会转化为当前时区，如 
            echo date("Y-m-d, H:i:s", 0);
            会输出：1970-01-01, 08:00:00


    2. 类似 PRC 这种时区只是为了兼容旧系统而保留，不推荐再使用 [2]

    
    3. 在 Laravel 中，时区在 php.ini 中设置后还不能生效。需要在 config/app.php 中设置 'timezone' => '';
