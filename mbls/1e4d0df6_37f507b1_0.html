mail函数

    bool mail ( string $to , string $subject , string $message [, string $additional_headers [, string$additional_parameters ]] )



mail 限制

    不支持 html 邮件和附件

    需要借助 STMP 发送邮件服务器来进行发送，并且不需要验证，而现在的商业邮件服务器都是需要进行验证的，因而只能自行配置不需要验证的 SMTP 服务器（如 Windows 下的 hmailserver）


还需要在 php.ini 中进行配置

    SMTP = localhost                      ; SMTP 服务器地址
    smtp_port = 25                        ; SMTP 协议默认端口号
    sendmail_from="bugber@localhost"      ; 在 windows 下需配置发件方的 email

<?php
 
$to = 'bugber@bugber.com';
 
$subject = "My Subject";
 
// The message
$message = "Line 1\r\nLine 2\r\nLine 3";
 
// In case any of our lines are larger than 70 characters, we should use wordwrap()
$message = wordwrap($message, 70, "\r\n");
 
 
// Send
if(mail($to, $subject, $message)){
    echo "success!";
} else {
    echo "failed!";
}
 
?>
