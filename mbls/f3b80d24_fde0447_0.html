命令
    每个命令都用分号（；）隔开，分号告诉 MySQL 执行这个命令。
    MySQL提示符和持续符号：>
    使用命令行前，确保将C:\mysql\bin\mysql放入path路径

大小写
        数据库被映射到具有某种文件结构的目录，而表则映射为文件
        SQL语句不区分大小写
        数据库和表的名称区分大小写或不区分（和OS一样）。
        列的名称和别名的名称不区分大小写，但同一SQL语句中不能混写
  
MySQL服务的停止(windows)
       NET STOP MySQL
       或 mysqladmin shutdown

连接到MySQL
       进入命令行，mysql -h hostname -u username -p
               -h, 指定主机，本机则可忽略
               -u, 指定用户，不指定则默认登陆操作系统的用户名，u后可以无空格
               -p, 告诉服务器要使用密码来连接mysql，执行后会提示输入密码。没有密码则忽略此项。
                       -p后面也可直接加密码，p后不要有空格
       登录时指定数据库：-D dbname           

退出mysql命令行
        exit or quit


运行现有的SQL文件

    > mysql -h host -u user_name -D dbname -p < someother.sql


基本DDL语句

        创建数据库
                mysql > CREATE DATABASE test1;

        显示系统中存在的数据库
                mysql > SHOW DATABASES;

        选择要操作的数据库
                mysql > USE test1;

        查看当前数据库中有哪些表
                mysql > SHOW TABLES;
                mysql > SHOW TABLES FROM db_name;

        获取该表中的所有列
                mysql > SHOW COLUMNS FROM db_name from table_name;
                mysql > SHOW COLUMNS FROM db_name.table_name;

        删除数据库
                mysql > DROP DATABASE test1;

        创建表
                mysql > CREATE TABLE tablename ( column_name_1  column_type_1  constraints,
                                                 column_name_2  column_type_2  constraints,
                                                 ...
                                                 column_name_n  column_type_n  constraints);
        
        查看表的定义
                mysql > DESC tablename;
                或用DESCRIBE亦可

        查看创建表的SQL语句
                mysql > SHOW CREATE TABLE tablename \G;              // \G 使得记录能够按照字段竖向排列
        
        删除表
                mysql > DROP TABLE tablename;

        创建索引
                主键会自动创建索引
                非主键的列，可以添加索引来改善性能
                        CREATE  [UNIQUE|FULLTEXT] INDEX index_name
                        ON table_name  ( index_column_name  [(length)]  [ASC | DESC], ...])
                 
                FULLTEXT索引用来索引文本字段
                length 字段允许指定只有该字段前length个字符将被索引
        
        查看用户所拥有的权限
                mysql > SHOW GRANTS for user_name;
