web 服务器代码管理方案

    方案一

        本地 git 用 source tree 或者直接命令 git push 到码云

        服务器上 git clone 或 git pull

        本地和与服务没有直接交互，都是通过码云


    方案二

        本地直接 ftp 到服务器



方案一补充

    服务器上还得装 composer， composer 的 vendor 目录，一般不会让 git 来管理。

    因为本地 push 到码云时，并不会把 composer 的 vendor目录 push 上去

    因而在服务器上 git clone 或 pull 后，需要再用 composer install 把对应的包安装了
