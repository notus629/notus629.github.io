伪静态配置 - 步骤

    环境： Windows 10, WAMPSERVER 3.0.6 64bit (Apache 2.4)


    1. 加载 Rewrite 模块

        对于 conf/httpd.conf 文件，找到
            // 默认为注释，去掉前面的注释“#”号
            LoadModule rewrite_module modules/mod_rewrite.so

    
    2. 若是目录级的 rewrite, 须允许在各目录中使用 .htaccess 文件

        // 在 httpd.conf 或 httpd-vhosts.conf 中，将 AllowOverride 的值改为 All (默认为 None)
        // 由于 Windows 中不允许建立 .htaccess 文件，在命令行下，使用 echo a > .htaccess 来建立。
        AllowOverride All

    
    3. 建立 rewrite 规则。服务器级在 httpd.conf 文件中，目录级则在相应目录（如根目录）下建立的 .htaccess 文件中，加入规则代码

        // 示例
        RewriteEngine on                                     // 启动rewrite引擎
        RewriteRule ^/index([0-9]*).html$ /index.php?id=$1   // “([0-9]*)” 代表范围 用(.*)代表所有，下同。
        RewriteRule ^/index([0-9]*)/$ /index.php?id=$1 [R]   // 虚拟目录
