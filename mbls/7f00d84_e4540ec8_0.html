02. Memcached安装演示(13')



安装步骤 - Windows

    1. 下载 memcached

    2. 进入 cmd, 切换到 memcached.exe 文件所在目录

    3. 命令行运行, memcached.exe -d install (d 表示 daemon)

    4. 启动 memcached

        a. 通过 Windows 服务
        b. 命令 memcached.exe -d start       ## 以后台方式启动
        c. 使用 netstat -an , 看到 11211 端口，说明启动成功。使用 netstat -anb 可以看到是哪个程序监听，还可以看到哪些用户连接到当前的服务器
