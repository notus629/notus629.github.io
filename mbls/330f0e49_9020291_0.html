phpize [1]

    The phpize command is used to prepare the build environment for a PHP extension.

## the sources for an extension are in a directory named extname
$ cd extname
$ phpize
$ ./configure
$ make
# make install

    安装完成后，会创建一个 extname.so 文件，并放入 php 的扩展目录 (extension directory)

    接下来，须手动在 php.ini 中加入： extension=extname.so



若系统没有 phpize 命令 [1]

    如果是使用已经预编译好的 packages，需要再安装合适的 php packages 的 devel 版，因为该版本通常会包含 phpize 命令，以及用来构建 PHP 和 PHP extension 的合适的头文件。
