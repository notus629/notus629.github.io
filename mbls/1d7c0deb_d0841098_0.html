集成安装环境列表



1. AppServer - PHP集成化安装包[1]

    AppServer: Apache + PHP + MySQL + phpMyAdmin

    PHP 网页架站工具组合包，AppServ 所包含的软件有：Apache、Apache Monitor、PHP、MySQL、phpMyAdmin 等

    For Windows Only.



2. XAMPP（Apache+MariaDB+PHP+PERL）[2]

    一个功能强大的建站集成软件包。这个软件包原来的名字是 LAMPP，但是为了避免误解，最新的几个版本就改名为 XAMPP

    Windows下的Apache+Mysql/MariaDB+Perl/PHP/Python, now use MariaDB instead of Mysql

    支持各国文字，支持 Windows/Linux/Mac OS

    XAMPP 只为开发者服务，不要用于生产环境。XAMPP MySQL 管理员(root)未设置密码



3. WampServer[3]

    集成了 Apache、PHP 和 MySQL 的开发套件，而且支持不同 PHP 版本、MySQL 版本和 Apache版本的切换。

    For Windows only.



4. LNMP一键安装包[4]

    用于 Linux 环境一键安装 LNMP 环境，同时该网站还包含有 LAMP 一键安装包

    For Linux only.



5. phpStudy

    国产，支持Apache/Nginx/Tengine/Lighttpd/IIS7/8/6

    支持Win2000/XP/2003/win7/win8/win2008/Linux 操作系统

    可以选择PHP版本，和java下有类似的JspStudy


6. visual amp/visual nmp

    类似 phpstudy, 随意切换版本

    a 表示 Apache, n 表示采用 Nginx



7. MAMP[5]

    My Apache - MySQL - PHP

    MAMP 免费，MAMP PRO 收费

    For macOS and Windows



8. EasyPHP

    EasyPHP 是一个 Windows 下的Apache+Mysql+Perl/PHP/Python开发包，包中集成了 PHP、Apache、MySQL，同时也集成了一些辅助的开发工具，如数据库管理工具，PhpMyAdmin 和 php调试工具 Xdebug，无需配置，就可运行。

    EasyPHP 是由法国人开发，经过 EasyPHP 整合后的Apache、MySQL及PHP精简很多，运行速度比独立安装的Apache、MySQL及PHP相对较快且比较稳定。

    还可以采用 Nginx 搭配 PHP7 作为 WebServer, 支持在 Nginx 和 Apache 之间切换



9. Laragon [6]

    功能包含    

        PHP 7.0.8. Laragon集成最新的PHP7.0.8，保持在对象集中的核心方法.它可以比CGI或者Perl更快速地执行动态网页。

        Composer & cmder. Laragon集成Composer、cmder和Git等工具，Composer是PHP的一个依赖管理工具。它允许你申明项目所依赖的代码库。

        Memcached & Redis. Memcached 是一个高性能的分布式内存对象缓存系统。Redis是一个开源的使用ANSI C语言编写、支持网络等。

        Sublime & notepad++. 还集成了Sublime、notepad++、FTP等工具，让您感觉到这功能集全还方便开发！
