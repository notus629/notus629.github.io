构建 GitHub Pages

    详细构建方式参见 [1]



步骤 (参见 [1] 中的向导） - Windows 10, Git Bash 环境下

    1. 建立一个 repository, 命名为: username.github.io (前面的 username 一定得是你的用户名或组织名）

    2. 选择你所使用的 git client, 这里选 A terminal

    3. clone the repository

    4. $ git clone https://github.com/notus629/notus629.github.io

    5. Enter the project folder and add an index.html file:
        $ cd username.github.io
        $ echo "Hello World" > index.html

    6. push it, 执行 push 命令后，会弹出登录 GitHub 窗口（SSH Key 没用了？）
        $ git add --all
        $ git commit -m "Initial commit"
        $ git push -u origin master


    默认 GitHub pages 页面 URL： http://username.github.io

    
    注意：页面显示有延时
