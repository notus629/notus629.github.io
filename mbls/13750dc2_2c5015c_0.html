yum 命令

    $ yum install pkg                          ## 安装某个软件

    $ yum check-update                         ## 检查某个软件的更新

    $ yum remove pkg                           ## 删除某个软件

    $ yum search pkg                           ## 搜索某个软件

    $ yum clean all                            ## 清除 yum 缓存

    $ yum update                               ## 更新所有已安装的软件

    $ yum update zlib                          ## 更新 zlib

    $ yum list zlib                            ## 显示 zlib 安装包

    $ yum upgrade                              ## 升级系统版本
