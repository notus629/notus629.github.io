JSON

    JavaScript 对象表示法 （JavaScript Object Notation JSON）   

    像对象字面量，但并不是对象

        {
            "location": "San Francisco, CA",
            "capacity": 270,
            "booking": true
        }



说明

    键：放在双引号中，非单引号

    每个键值对都用逗号隔开



值的类型

    string        文本（必须放在引号中）

    number        数字

    Boolean       ture or false

    array         数组。可由值组成，也可以由对象组成

    object        JavaScript 对象可以包含任何子对象或数组

    null          表示值为空或已丢失



JSON 格式转换

    JSON.stringify()      序列化，将 JavaScript 对象转换成 JSON 格式的字符串

    JSON.parse()          反序列化，将 JSON 数据（字符串）转换成浏览器可使用的 JavaScript 对象
