Event 函数

    abort      图像加载中断时触发事件

    blur       元素失去焦点时触发事件

    change     当内容改变时触发事件

    click      当左键点击因素时触发事件

    dblclick   当左键双击因素时触发事件

    error      加载文档或图像发生错误时触发事件。

    focus      元素获得焦点时触发事件

    keydown    某个键盘按键被按下时触发事件

    keyup      某个键盘按键被松开时触发事件

    keypress            某个键盘按键被按下并松开时触发事件

    load            一个页面或一幅图像加载完成时触发事件

    mousedown            当鼠标按钮被按下时触发事

    mousemove            鼠标指针在绑定元素上方移动时触发事件

    mouseout            鼠标指针移出指定的元素时触发事件

    mouseover            鼠标指针移动到指定的元素上时触发事件

    mouseup            鼠标按键被松开时触发事件

    reset            表单中的重置按钮被点击时触发事件

    resize            在调整窗口或框架大小时触发事件

    select            在文本框中的文本被选中时触发事件

    submit            在表单被提交时触发事件

    unload            在用户退出页面时（关闭浏览器或者刷新页面）触发事件

    input            检测用户的输入状态

    keyCode            监听键盘按键触发相应事件

    target            获得触发事件的元素

    stopPropagation()        禁止事件冒泡

    button                在鼠标点击时返回一个值

    preventDefault()        取消事件的默认动作。

    addEventListener()        事件监听处理函数
