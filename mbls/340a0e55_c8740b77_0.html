TypeScript & CoffeeScript & Babel



CoffeeScript 

    CoffeeScript 是一套JavaScript的转译语言

    CoffeeScript 会将类似 Ruby 语法的代码编译成 JavaScript，而且大部分结构都相似，但不同的是 CoffeeScript 拥有更严格的语法。

    参见 [7], 推荐 TypeScript

    截止 2017-5-3，GitHub 13.9k stars



TypeScript [1][2][3]

    微软开发， JavaScript 的一个超集, 他支持所有的 JavaScript 语法。并在此之上对 JavaScript 添加了一些扩展，如 class / interface / module 等。这样会大大提升代码的可阅读性。

    任何现有的 JavaScript 程序可以不加改变的在 TypeScript 下工作。

    由于他是 JavaScript 的超集，对 JavaScript 开发者来说入门门槛很低（相对于 Dart / CoffeeScript 等其他 JavaScript 变种来说 ）。

    TypeScript 作为 ECMAScript 的子集

    截止 2017-5-3, GitHub 21.6k stars



Babel [6]

    默认情况下，Babel 自带了一组 ES2015 语法转化器。这些转化器能让你现在就使用最新的 JavaScript 语法，而不用等待浏览器提供支持.

    Babel 内嵌了对 JSX 的支持。通过将 Babel 和 babel-sublime 包（package）一同使用可以让源码的语法渲染上升到一个全新的水平。

    Babel 是一个转换编译器, 它能将 ES6 转换成可以在浏览器中运行的代码



ECMAScript 

    ECMAScript 6 is the newest version of the ECMAScript standard.

    ES2015 是对 2009 年标准化的 ES5 的更新
