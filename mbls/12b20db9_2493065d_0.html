Installation

    ## or download http://download.redis.io/releases/redis-stable.tar.gz
    $ wget http://download.redis.io/releases/redis-3.2.8.tar.gz
    $ tar xzvf redis-3.2.8.tar.gz
    $ cd redis-3.2.8
    $ make



Run

    The binaries that are now compiled are available in the src directory. Run Redis with:

    $ src/redis-server




interact with built-in client

    $ src/redis-cli
    redis> set foo bar
    OK
    redis> get foo
    "bar"
