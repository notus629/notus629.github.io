NULL 值

    There is only one value of type null, and that is the case-insensitive constant NULL.



A variable is considered to be null if:

    it has been assigned the constant NULL.

    it has not been set to any value yet.

    it has been unset().



Note:

    Casting a variable to null using (unset) $var will not remove the variable or unset its value. It will only return a NULL value.

    A null character ("") is not equivalent to the PHP NULL constant.


    Empty array is converted to null by non-strict equal '==' comparison.

        $a = array();
        $a == null  <== return true
        $a === null < == return false
        is_null($a) <== return false
