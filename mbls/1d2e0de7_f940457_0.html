PHP 的不同执行方式

    运行方式

        CGI, fastCGI, SAPI, ISAPI


    版本

        TS, NTS

        debug pack版： 给 C++ 程序员开发或修改 PHP 扩展用的   


SAPI方式

    对 IIS 应该就是 ISAPI

    对 Apache 就是 Apache 模块，在 Windows 下也会使用 ISAPI

    多线程方式

    使用 TS 版



CGI 

    是多线程，基本上应该是不再使用



fastCGI 

    是多进程，保持在内存中，速度快，占用资源多

    使用 NTS 版



Nginx 与 Apache

    Nginx 使用 fastcgi 多进程方式，因而使用 NTS（非线程安全）版

    Apache 使用 sapi 的多线程方式，因而使用 TS（线程安全）版
