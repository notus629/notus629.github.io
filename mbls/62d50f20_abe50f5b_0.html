IE5~IE8中的事件对象

    不会自动传递给事件处理函数/监听器函数

    需要在事件处理函数中，使用 window.event 来获取（即事件对象作为 window 的属性而存在）

<script type="text/javascript">
    function checkUsername(e){
        // 未获取到事件对象，表明是 IE5~IE8 浏览器
        if(!e){
            e = window.event;
        }
 
        // 利用短路特性兼容 IE5~IE8
        var target = e.target || e.srcElement;
    }
</script>
