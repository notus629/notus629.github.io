Apache 伪静态 Rewrite

    Rewrite 主要的功能就是实现 URL 的跳转，采用基于 Perl 风格的正则表达式 (PECL)

    可基于服务器级的(httpd.conf)和目录级的 (.htaccess)两种方式

    使用 mod_rewrite 模块    



Rewrite 模块的安装或加载

    使用 Rewrite 须先安装或加载 rewrite 模块。方法有两种

        一种是编译 Apache 的时候就直接安装 rewrite 模块

        另一种是编译 Apache 时以 DSO 模式安装 apache, 然后再利用源码和 apxs 来安装 rewrite 模块。（即 loadmodule）
