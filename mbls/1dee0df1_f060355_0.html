查看是否已经安装 iptables

    $ iptables -V



安装 iptables

    ## centos
    $ yum install iptables



打开配置文件(centos)

    $ vi /etc/sysconfig/iptables



查询防火墙是否有打开 80 端口

    $ /etc/init.d/iptables status



注：

    CentOS 7 默认使用 firewalld 来管理 iptables 规则


# systemctl status firewalld 查看服务状态

# systemctl stop firewalld.service 关闭服务
