cat /proc/version  

    查看 /proc 下的 version 信息，即当前系统的内核版本号及系统名称



set                

    Manipulate shell variables and functions

    With no options and arguments; displays the names and values of all shell variables and functions.

    [Ref 1] http://ss64.com/bash/set.html



du (disk used)     

    功能：查看磁盘剩余空间（计算每个文件的磁盘用量，目录则取总用量。）

    a) 查看文件夹下所有文件大小（包括子文件夹） 

        du –h        # 当前文件夹
        du –h ftp     # 指定文件夹

    b) 查看指定文件大小

        du –h ./test.c # 指定文件大小
        du –hs ftp    # 该目录整个所占空间，s (summarize)



df (disk free)     

    功能：Linux 系统以磁盘分区为单位查看文件系统，可以加参数查看磁盘剩余空间信息

    df –hl # 查看硬盘容量

    df –k  # 以块为单位的容量



wget 命令          

    GNU Wget is a free software package for retrieving files using HTTP, HTTPS and FTP, the most widely-used Internet protocols. 

    It is a non-interactive commandline tool, so it may easily be called from scripts, cron jobs, terminals without X-Windows support, etc.

    开源软件：可以在linux和windows上使用，

    基本的语法是：wget [参数列表] URL

    同时支持FTP和HTTP下载方式；尽管现在大部分软件可以使用HTTP方式下载
