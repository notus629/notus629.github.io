CSS属性动画



animate() 方法

    通过改变 CSS 属性来创建自己的动画特效。

    可以表现为数字的 CSS 属性可以用来创建动画。

    CSS 属性转换为 camel 命名法， 如 background-color 命名为 backgroundColor.

    CSS 属性使用对象描述标记进行表示。



格式

    .animate({
        // Styles you want to change
        }[, speed][, easing][, complete]);


    speed       表示动画的持续时间（以毫秒为单位），可使用关键字 slow, fast

    easing      linear 线性速度， swing 开始结束时慢，中间快。

    complete    回调函数。动画结束时要调用的函数。
