## 3 步曲
$ ./configure
$ make           ## or " make && sudo make install " together
$ sudo make install



./configure                      

    利用 configure 脚本来进行配置，根据用户提供相关参数生成对应的 makefile 文件

    一般可以用 $ ./configure --help for full options

    --prefix 选项，指定软件安装的位置，不指定则按默认路径
    或借助安装文档 README 或 INSTALL 来推断应该设置哪些选项

    $ ./configure --prefix=/usr/local/games/foobillard

    configure 脚本首先检查当前系统是否符合编译条件，系统应该安装有正确的编译器（gcc）


make                           

    make 是一种高级编译工具，可以依据 makefile 文件中的规则调用合适的编译器编译源代码

    $ make                      ## 编译源代码
    $ sudo make install         ## 编译完源代码后，此命令用来安装软件，需 root 权限
