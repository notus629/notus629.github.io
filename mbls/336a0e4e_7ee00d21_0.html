返回登录结果（还包含一次慕课与QQ的交互）

    有了 code 之后，基本确定登录成功，但为了确保 code 是合法获取到的，慕课网须再使用

        User Authorization URL (用户授权的令牌请求服务地址）

        用户登录授权之后需要请求的一个带有特定参数的 URL

            https://xxx.qq.com/oauth/...?...
            &client_id=100490398&......
            &client_secret=xxxxxxxx&....
            &code=xxxxxxxxxx......

        client_id, client_secret 即第三方网站（慕课网）从 QQ 得到的 appid 和 key.

        code 授权后拿到的 code 参数，有时间限制, 几秒到 1 分左右。只可以被使用 1 次


    最后再返回一个响应： xml 或 json 的等格式

        里面会包含用户的昵称、头像等基本信息。

        响应参数里还包含：AccessToken 即用户通过第三方应用访问 OAuth 接口的令牌（这样慕课网的内容可以通过 QQ 的接口进行分享等其它操作。）
