z-coding or zen-coding?

    用 CSS 的规则在 HTML 文档里快速键入内容


    举例：

        ul.nav>li*5>a，然后按Tab键，马上能出来一大段代码，如下所示：  
            <ul class="nav">
                <li><a href=""></a></li>
                <li><a href=""></a></li>
                <li><a href=""></a></li>
                <li><a href=""></a></li>
                <li><a href=""></a></li>
            </ul>


    再按 Tab 键可以跳到两个标签中间



script:src

    <script src='光标'></script>



#idvalue 或 div#idvalue

    <div id="idvalue">光标</div>
