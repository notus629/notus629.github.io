Bit-Value Type - BIT [1]


TypesValue rangeDescription
BIT[(M)]M can range from 1 to 64BIT(M) enables storage of M-bit values

To specify bit values, b'value' notation can be used. 

b'111' and b'10000000' represent 7 and 128, respectively
