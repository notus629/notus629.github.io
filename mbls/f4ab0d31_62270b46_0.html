Ubuntu 系统安装


虚拟机安装ubuntu server 16.04中文版时出现“无法安装busybox-initramfs”

    网上的解决办法不管用，这应该是中文版专有的bug, 解决方法就是安装英文版，或者安装14.04的中文版。

    Leo 实战：第一次选语言选English, 第二次再选中文，就没有问题了。

    仍然在控制台中没办法显示中文，中文显示为乱码。
