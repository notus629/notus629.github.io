安装错误："以一种访问权限不允许的方式做了一个访问套接字的尝试"？ [1]

    错误背景：安装 Win10 更新后出现。

    错误原因：端口冲突，TCP80 端口已被占用。

    解决方案 1：将 Apache 的配置文件  Apache 安装目录下的 conf/httpd.conf，Listen 80，即 Apache 的监听端口改为其它值（如 88）即可。

    解决方案 2：即停止占用 port 80 的服务，经 netstat -ano 查询得知占用此端口的 PID 为 4，进一步，为 NT Kernel & Syste 服务，google之，可能使用此服务的有：

        World Wide Web Publishing Service
        Web Deployment Agent Service
        SQL Server Reporting services（最终此服务终止后解决问题，可能此服务即由 Win10 更新后开启）
