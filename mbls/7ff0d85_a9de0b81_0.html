    $ yum install memcached    



memcached （即 memcached server) 安装, from latest source code 

    首先需要安装依赖库 libevent [3]

        Debian/Ubuntu: $ apt-get install libevent-dev 
        Redhat/Centos: $ yum install libevent-devel


    然后，下载、解压、转到 Source 所在文件夹、配置、编译、安装 [1,4]

Install memcached on Windows [2]

    Memcached works on most Linux and BSD like systems. There is no official support for windows builds.[4]
    

    Installation of memcached < 1.4.5 (可以以服务的形式安装)

        1. 安装，打开 cmd, 进入 memcached.exe 目录
        > c:\memcached\memcached.exe -d install

        2. 启动和关闭
        > c:\memcached\memcached.exe -d start
        > c:\memcached\memcached.exe -d stop

        3. 修改 memcached 配置，运行 regedit, 定位到
        "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\memcached"
        若要修改内存限制，将 ImagePath 值改为：
        "c:memcachedmemcached.exe" -d runservice -m 512
        除 "-m 512" 参数外，还可以添加其它参数到路径中，运行 "c:\memcached\memcached.exe -h" 可以查看参数列表。

        4. 卸载 memcached 服务
        > c:\memcached\memcached.exe -d uninstall


    Installation of memcached >= 1.4.5 (不能以服务形式运行) 

        1. 必须用任务调度程序 (task scheduler) 作为一个常规的进程来运行。如果需要配置在 Windows 启动时自启动，需要以管理员权限打开 cmd, 运行下面命令
        > schtasks /create /sc onstart /tn memcached /tr "'c:\memcached\memcached.exe' -m 512"
    
        2. 移除 memcached 任务计划
        > schtasks /delete /tn memcached
