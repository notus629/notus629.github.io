第7章 jQuery

   

使用 CSS 样式选择器来查找元素： jQuery() 或简写 $()

    参数： CSS样式选择器

    返回 jQuery 对象    



jQuery 方法来操作元素

    $('li.hot').addClass('complete');



优点：

    跨浏览器

    使用 css 样式选择器语法，选择元素更简单，更精确

    事件处理更简单，兼容浏览器

    方法会应用于所有元素，无需遍历

    动画方法

    可在选择的元素上应用多个方法

// 选中所有的 h1~h6 标头，将 'headline' 添加到 class 属性中
$(':header').addClass('headline');
 
// 选中前三个列表项(索引0, 1, 2)，隐藏这些元素，淡入显示到视图中
$('li:lt(3)').hide().fadeIn(1500);
 
// 在每个 li 上设置一个事件监听器，点击其中一个列表时，触发一个匿名函数，将元素中页面中移除
$('li').on('click', function(){
    $(this).remove();
});
