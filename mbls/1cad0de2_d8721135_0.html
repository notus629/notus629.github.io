单字母函数


        I 函数

                用于更加方便和安全的获取系统输入变量，可以用于任何地方

                格式
                        I('变量类型.变量名',['默认值'],['过滤方法']);

变量类型是指请求方式或者输入类型，包括：

变量类型含义
get获取GET参数
post获取POST参数
param自动判断请求类型获取GET、POST或者PUT参数
request获取REQUEST 参数
put获取PUT 参数
session获取 $_SESSION 参数
cookie获取 $_COOKIE 参数
server获取 $_SERVER 参数
globals获取 $GLOBALS参数
注意：变量类型不区分大小写。
变量名则严格区分大小写。
默认值和过滤方法均属于可选参数。


                用法

                        echo I('get.id');                                           // 即 $_GET['id'];
                        echo I('get.id', 0);                                       // 默认值，如果不存在，则返回0
                        echo I('get.id', htmlspecialchars');                // 采用htmlspecialchars对$_GET['id']进行过滤，不存在则返回空字符串。
                        I('get.');                                                      // 获取整个$_GET数组 
                        I('id');                                                         // param简写法，等同于I('param.id');

                param变量特殊用法

                        param可用数字索引的方式获取URL参数（必须是PATHINFO模式参数有效，无论是GET还是POST方式都有效）

                        URL地址：http://serverName/index.php/New/2013/06/01

                        echo I('param.1');                // 输出 2013
                        echo I('param.2');                // 输出 06
