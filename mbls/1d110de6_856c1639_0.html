如果是在本机上

    > netstat -an | find "port_no"



若是在外部机器上

    Windows:
        > telnet host(ip or domain) port

        ## 示例： 测试远程主机 10.8.28.100 的 3306 端口是否打开
        ## 按 ctrl + F5 终止
        > telnet 10.8.28.100 3306
            

    Linux:
        $ telnet host:port



Win7 之后默认 telnet 服务关闭

    打开 telnet 服务

        控制面板 -> 程序和功能 -> 启动或关闭 Windows 功能 -> telnet 客户端 (Win10）

    

Windows 自带图形界面侦听端口工具

    控制面板->管理工具->资源监视器
