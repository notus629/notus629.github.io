使用 cURL 完成简单的请求主要分为以下四步

    1.初始化，创建一个新 cURL 资源

    2.设置 URL 和相应的选项

    3.抓取 URL 并把它传递给浏览器(发送请求和接收服务器数据)

    4.关闭 cURL 资源，并且释放系统资源



基本过程如下

    $ch = curl_init();           // 创建 cURL 资源
    curl_setopt($ch, ..);        // 设置 URL 和相应的选项
    curl_exe($ch);               // 发送请求和接收服务器数据
    curl_close($ch);             // 关闭 cURL 资源，并且释放系统资源
