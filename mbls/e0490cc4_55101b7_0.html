MySQL 数据库管理工具

    MySQL 监视器(命令行)
        与 MySQL 交互的命令行客户端工具，MySQL 自带

    phpMyAdmin：基于浏览器的工具，php 用于管理 mysql 数据库的工具

    workbench: MySQL 自带的客户c/s应用

    企业一般使用：navicat

    SQLyog Ultimate： 类似 navicat 的图形化工具，排名低于 navicat



MariaDB

    A fork of the MySQL , MariaDB is kept up to date with the latest MySQL release from the same branch and in most respects MariaDB will work exactly as MySQL. All commands, interfaces, libraries and APIs that exist in MySQL also exist in MariaDB.
