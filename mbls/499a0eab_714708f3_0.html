git branch & git checkout



git branch - 显示分支一览表

    ## 将分支名列表显示，同时可以确认当前所在的分支
    ## 当前所在分支前面显示 * 号
    $ git branch 



创建、切换分支

    git checkout -b branch_name      以当前 master 分支为基础，创建新的分支并切换到新分支

    git checkout branch_name         切换到 branch_name 分支

    git checkout -                   切换到上一分支


    git branch branch_name           创建新分支，但当前分支不变

    git branch -a                    查看当前分支的相关信息，同时显示本地仓库和远程仓库的分支信息


    示例

        ## 创建名为 freature-A 的分支
        ## 并且将当前分支切换到 feature-A
        $ git checkout -b feature-A

        ## 创建 feature-A 分支
        $ git branch feature-A
        ## 切换到 feature-A 分支
        $ git checkout feature-A



当前分支

    git add & git commit 会提交到当前分支
