将本地代码推送到 GitHub 新 repo

   1. 在 GitHub 上创建新的仓库(空仓库)。为了避免发生合并冲突，在初始化你的仓库时，请不要创建 README, license 或 gitignore文件。


    2. 打开终端，并切换到代码的根目录. 初始化本地 git 仓库 

        $ git init
        $ git add .  

        # 如果要从仓库中移除 
        $ git reset HEAD 你的xxx文件 

        # 如果要从仓库移除目录 
        $ git reset HEAD -- 你的xxx目录/

        # 第一次 commit 
        $ git commit -m "第一次提交"


    3. 获取 GitHub 仓的 URL，一般在仓库的首页

        ## 在终端里，为你的本地仓库，添加一个远程仓库 URL 
        ## 从本地推送到远程，需要事先特别申明 origin
        $ git remote add origin 远程GitHub仓的URL

        ## 验证新的远程 URL 
        $ git remote -v

        ## 将本地的代码，推动到远程的 GitHub 
        $ git push origin master
