4. 如下例中，在html中出现的xmlns属性是什么意思？

    <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
    <html xmlns="http://www.w3.org/1999/xhtml">

答：xmlns是XHTML namespace的缩写。

    作用：由于xml允许定义自己的标记，但你定义的标记和其他人定义的标记有可能相同，但表示不同的含义。当文档交换或者共享的时候就容易产生错误。为避免这种错误产生，xml采用声明，允许你通过一个网址来识别你的标记。名字空间    

    XHTML是HTML先xml过度的标记语言，它需要符合规则，因此，也需要定义，又因为XHTML1.0不能自定义标识，所以，它的都相同，就是：http://www.w3.org/1999/xhtml名字空间xml文档名字空间

    Leo注：现在html网页（2016-11-6）很少有加这个属性，似乎没多大必要?


5. 在当前窗口中打开，在新窗口中打开？

答：<base target="_self"/>，默认方式，即本页刷新
      <base target="_base" />, 在新窗口中打开
      <a href="xxx.htm" target="_blank">超链接内容</a>, 特定元素中在新窗口中打开
      
_blank
在新窗口中打开被链接文档。
_self
默认。在相同的框架中打开被链接文档。
_parent
在父框架集中打开被链接文档。
_top
在整个窗口中打开被链接文档。
framename
在指定的框架中打开被链接文档。
