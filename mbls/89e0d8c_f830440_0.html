/ 根目录

    /bin      

        进行系统操作所需要的基础命令，即最小系统所需要的命令

        目录中的文件都是普通用户可以使用的可执行文件，如 sl, cp, mkdir等命令


    /boot     

        Linux 的内核及引导系统程序所需要的文件

        如 vmlinuz, initrd.img 等文件

        系统引导管理器 GRUB 
       

    /dev

        设备文件存储目录，如磁盘、光驱、USB 接口、声卡、终端设备等


    /etc

        系统配置文件所在地, 如 

            /etc/apt            目录, 存放 apt 工具的配置文件
            /etc/init.d         目录, 用来存放系统或服务器的启动脚本
            /etc/network        目录，存放一些网络管理的配置文件
            /etc/resolv.conf    目录，配置域名信息
            /etc/resolvconof    目录，配置域名信息
            /etc/ssh            目录，存放SSH客户端及服务器的配置文件
            /etc/X11            目录，存放X-Window相关的配置文件

    /lib

        库文件的存放目录


    /lost+found

        在ext2, ext3, ext4文件系统中，意外崩溃或意外关机而产生的碎片        


    /media

        即插即用型存储设备的挂载点自动在这个目录在创建，如USB, CDROM/DVD(注新版Ubuntu直接有一个/cdrom目录了）            


    /mnt

        用作挂载点的父目录，将其他储存设备挂载到该目录的子目录中


    /opt

        可选的意思，有些软件包会被安装在这里，建议将自编译的软件安装在此目录


    /proc

        操作系统运行时，进程信息及内核信息（如CPU、硬盘分区、内存信息等）放在这里， 如

            /proc/cpuinfo       文件，保存了关于处理器的信息，类型、厂家、型号和性能等
            /proc/devices       文件，保存了当前运行内核所配置的所有设备清单
            /proc/dma           文件，保存了当前正在使用的 DMA 通道
            /proc/filesystems   文件，保存了当前运行内核所配置的文件系统
            /proc/interrupts    文件，保存了当前正在使用的中断和曾经有多少个中断等信息
            /proc/ioports       文件，保存了当前正在使用的I/O端口信息


    /root

        root 用户的家目录


    /sbin

        与 /usr/sbin, usr/local/sbin 目录一样，存放的都是涉及系统管理的必须有 root 权限才能执行的命令


    /tmp

        临时文件目录，与 /var/tmp 目录类似


    /usr

        存放系统软件包程序的目录，如命令、帮助文件等。Linux 发行版官方提供的软件包大多安装在这里，配置则在 /etc 目录中， 如

            /usr/share          系统中共用的文件的存放目录
            /usr/bin            普通用户可执行文件的存放目录
            /usr/local/bin      同上，有时 /usr/bin 中的文件为 /usr/local/bin 的链接文件
            /usr/sbin           超级权限用户 root 的可执行命令的存放目录
            /usr/local/sbin     同上
            /usr/include        程序的头文件存放目录
            /usr/local          目录，用户自编译安装软件的默认存放目录，类似 windows 下的 program files


    /var

        内容是经常变动的，如

            /var        存放系统日志的目录
            /var/www    Apache 服务器的站点存放目录
            /var/spool  打印机、邮件、代理服务器等假脱机文件（输出井和输入井）存放目录
