jScript

    *.js 文件 

    ECMAscript 的 windows 实现， 与javaScript 略有差异

    可用 wscript.exe or cscript.exe 来运行（windows 10 不支持用 wscript 来运行 js 文件）

        实际在 windows 10 中执行 cscript a.js 提示信息就是显示 Microsoft @ Window Based Script Host

    wscript.exe, cscript.exe 路径： C:\Windows\System32

    wscript: windows script host

    cscript: Microsoft @ Console Based Script Host

    wscript: Microsoft @ Window Based Script Host


命令行输出

    WSH.Echo("Hello world");

    // 或
    alert = function foo(s){WScript.Echo(s)}
    alert("hello world");


打开记事本

    function test1(){
        var wsh = new ActiveXObject('WScript.shell');
        wsh.run('notepad.exe');
    }
    test1();


总结：

    1. ie 内核可用，其它不行

    2. 本地可以，但默认会转出提示：是否允许阻止的内容

    3. 从服务器接收的网页不行
