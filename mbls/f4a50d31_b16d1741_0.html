附A 笔记中标记说明
    带下划线标记表示不推荐使用，应由css元素来替代。
        <input /> stands for html markup or css property
        height denotes element property or feature



重点：

    1.        class 特性

        如果你要指定一个元素属于几个不同的类，则用空格将它们隔开，如:

        <p class="important admmitance">...</p>


    2.   块级元素主要有
        
        <div>, <h1>, <p>, <ul>, <li>


    3.        内联元素主要有
   
        <span>, <img>, <a>, <b>, <em>
