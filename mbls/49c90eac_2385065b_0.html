Redis 数据操作 （命令）

    1. 开启 Redis 服务
        $ redis-server xxxx.conf            ## 开启时加载配置文件

    2. 开启终端（客户端）
        Redis 安装目录下包含有 src/redis-cli
        ## 先进入 src 目录
        $ redis-cli
   
    3. 设置缓存值
        > set singwa 12

    4. 获取缓存值
        > get singwa 

    5. 设置过期时间
        ## 12 秒过期
        > setex singwa 12 ssssfff

    6. 删除缓存
        > del singwa



PHP 操作 Redis

    1. 安装 phpredis 扩展

    2. php 连接 redis 服务 - connect (127.0.0.1, 6379);

    3. set/setex 设置缓存

    4. get 获取缓存
