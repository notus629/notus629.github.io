git log - 查看提交日志

    git log 命令，可查看以往仓库提交的日志，即何人在何时进行了提交或合并，及操作前后的差别

    只能查看以当前状态为终点的历史日志


    ## 输出结果的 commit 行： 指向这个提交的哈希值
    ## Author 行：我们给 Git 设置的用户名和邮箱
    ## Date 栏：提交执行的日期和时间
    $ git log


    ## 仅显示提交信息的第一行
    $ git log --pretty=short


    ## 只显示指定文件或指定目录下的日志
    $ git log file_name
    $ git log directory_name


    ## 显示提交带来的改动（所有 commit）
    $ git log -p
    ## 只查看 README.md 文件的提交日志及提交前后的差别
    $ git log -p README.md


    ## 以图表形式输出提交日志
    $ git log --graph 


    ## 查看当前仓库执行过的操作的日志
    $ git reflog
