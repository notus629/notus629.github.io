2.        gvim & vim

3.        gvim, gvim easy, gvim read only
Windows下安装vim7.4(windows 10 64bit)后，桌面出现3个快捷方式，分别是gVim7.4, gVim Easy 7.4和gVim Read only 7.4.
Easy启动的时候是insert模式，适合普通windows用户的习惯；
Read-Only的用途：比如用read-only打开已经用vim打开过的文件，就不会提示让人烦躁的.swp文件存在的问题；比如可以用它来看TXT的电子书不会存在误删误改（ctrl+b ctrl+f很方便）

4.        vim 支持JavaScript编程
vim默认似乎不支持js库配色。
在vimfiles\syntax下放入jquery.vim，在vimrc中加入如下代码：
au BufRead,BufNewFile *.js set syntax=jquery

7.        set font in vim last line mode
In gvim, you can change the font using the Edit menu, Select Font. An alternative is to enter the command:
:set guifont=*
Once you have a font you like, you want to make it the default in the future. Do
:set guifont?
and Vim will display something like
guifont= Consolas:h14:cANSI
Alternatively, enter the following to insert the current font setting into the buffer:
:put =&guifont
Now put a line in your vimrc to set guifont to this value, like this:
if has('gui_running')
 set guifont= Consolas:h14:cANSI
endif
If there is a space in the font name, such as
guifont=Monospace 10
it is necessary to escape the space
       set guifont=Monospace\ 10
Ref: http://vim.wikia.com/wiki/Change_font
