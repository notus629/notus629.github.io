常见属性

    imgElement.src                          // 可以给图片src属性赋值

    someElement.value                       // 得到值，特别对于文本框、密码框等。也可以赋值

    event.keyCode                           // 按键的keycode（键码）, event为事件函数传递过来的事件对象

    window.location                         // 直接可以赋值，则把用户带到一个新地址
      
    windows.parent                          // 返回父窗口
        
    screen.width                            // 屏幕宽度



window对象

    默认如果属性或函数前不加对象名就指引用 window，但不推荐



style 对象
    
    使用style对象的属性的语法

        document.getElementById("id").style.property = "值";
            
        e.g. Object.style.visibility = visible|hidden|collapse;
