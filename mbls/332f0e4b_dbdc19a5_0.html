项目根目录下的 public 目录

    CSS， images, js 等静态资源



项目根目录下的 template 目录

    放置模板文件，如, singwa.php，里面都是 html 的结构



项目根目录下 index.php 文件

    当执行或请求此文件时，在网站根目录下，生成一个纯静态的 index.shtml 文件

    此静态文件 index.shtml 文件最终给用户访问

    用到 template/singwa.php 中的结构，但数据不同



数据来源

    从数据库或从 Redis / memcached 等缓存中获取



index.php 功能

    1. 连接数据库，从数据库里获取数据

    2. 把获取到的数据填充到模板文件里

    3. 需要把动态的页面转换为静态的页面，生成纯静态化文件
