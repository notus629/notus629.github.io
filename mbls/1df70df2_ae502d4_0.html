Vagrant

    Vagrant 就是一个普普通通的装了一个 Linux 的 VirtualBox 虚拟机，配以 vagrant 团队为之开发的一系列套件，辅助完成诸如安装初始化、文件同步、ssh、部署环境升级、功能插件安装等等一些列问题的开发环境部署套件。

    Vagrant provides the same, easy workflow regardless of your role as a developer, operator, or designer. 

    It leverages a declarative configuration file which describes all your software requirements, packages, operating system configuration, users, and more.

    Vagrant works on Mac, Linux, Windows, and more.

    Vagrant also integrates with your existing configuration management tooling like Chef, Puppet, Ansible, or Salt, so you can use the same scripts to configure Vagrant as production.
