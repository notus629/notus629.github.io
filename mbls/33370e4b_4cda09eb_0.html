LNMP 安装目录

    默认网站目录 : /home/wwwroot/default/


    Nginx 目录: /usr/local/nginx/

    Nginx 日志目录：/home/wwwlogs/


    MySQL 目录 : /usr/local/mysql/

    MySQL 数据库所在目录：/usr/local/mysql/var/

    PHPMyAdmin 目录 : /home/wwwroot/default/phpmyadmin/ 


    PHP 目录 : /usr/local/php/
