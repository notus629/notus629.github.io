有序集合

    存储键值对，即字符串成员与浮点分值之间的有序映射

    元素的排列顺序由分值大小决定

    有序集合的键被称为成员(member), 每个成员各不相同

    有序集合的值被称为分值(score), 必须为浮点数

    Redis 里唯一一个，既可以根据成员访问元素，又可以根据分值及分值的排序来访问元素的结构


    操作

        添加、获取、删除单个元素

        根据分值范围或者成员来获取元素



有序集合命令

    ZADD        将一个带有给定分值的成员添加到有序集合里面

    ZRANGE      根据元素在有序排列中所处的位置，从有序集合里面获取多个元素

    ZRANGE      获取有序集合在给定分值范围内的所有元素

    ZREM        如果给定成员存在于有序集合，则移除该成员

    ZINCRBY     对成员的分值执行自増操作（増加值可以设置）



举例

    >>> zrange zset-key 0 -1 withscores              ## 获取所有元素（包含分值）
    >>> zrange zset-key 0 -1                         ## 获取所有元素（仅显示成员）
