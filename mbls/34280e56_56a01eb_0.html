Alias

    Provides mapping from a URI to a directory - usually a directory outside of your DocumentRoot.

    If you are creating an Alias to a directory outside of your DocumentRoot, you may need to explicitly permit access to the target directory.

        Alias "/image" "/ftp/pub/image"
        <Directory "/ftp/pub/image">
            Require all granted
        </Directory>


    If the Alias directive is used within a <Location> or <LocationMatch> section the URL-path is omitted, and the file-path is interpreted using expression syntax. (Apache 2.4.19 and later)
<Location "/image">
    Alias "/ftp/pub/image"
</Location>
<LocationMatch "/error/(?<NUMBER>[0-9]+)">
    Alias "/usr/local/apache/errors/%{env:MATCH_NUMBER}.html"
</LocationMatch>
