require

    Apache2.4 目录访问控制指令 require

    Apache2.2 相关指令与 Apache2.4 require 指令不同，参见 [1]



实例 1 - 所有请求都被拒绝

    ## Apache2.4, 拒绝所有   
    Require all denied  


    ## Apache2.2 配置
    ## 排序，先拒绝后允许
    Order deny,allow    
    ## 拒绝所有
    Deny from all     



实例 2 - 所有请求都被允许

    ## Apache2.4, 允许所有
    Require all granted


    ## Apache2.2 配置
    ## 排序，先允许后拒绝
    Order allow,deny   
    ## 允许所有
    Allow from all     



实例 3 - splaybow.com 所有请求都被允许，其他拒绝

    ## Apache2.4 , 允许 splaybow.com
    Require host splaybow.com


    ## Apache2.2 配置:
    # 排序，先拒绝后允许
    Order Deny,Allow          
    # 拒绝所有
    Deny from all             
    # 允许splaybow.com
    Allow from splaybow.com   




常见访问控制指令

    # 允许所有来源访问
    Require all granted     

    # 拒绝所有来源访问
    Require all denied      

    # 允许表达式为 true 时访问
    Require expr expression 

    # 允许特定 IP 段访问，多个段之前用空格隔开。每个段使用开头几项表示
    Require ip 10 172.20 192.168.2 

    # 只允许来自域名 splaybow.com 的主机访问
    Require host splaybow.com
