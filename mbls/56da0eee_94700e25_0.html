js 中 6 种基本类型
       
    String

        如果一个变量或对象的属性包含一个字符串，可以使用String对象的方法
   
    
    Number

        如果一个变量或对象的属性存储了一个数字，可以使用Number对象的属性或方法
       

    Boolean

        有一个Boolean对象，很少被用到
       

    undefined

        变量被声明未被赋值，没有对象
       

    null 值

        没有值的变量，可能曾被赋值，但现在没有值，不具有对象


    Object

        数组和函数被认为是对象的类型
 
        数组也具有属性和方法
      

    前 5 种：简单，基本数据类型

       
    第 6 种：对象，复杂数据类型
