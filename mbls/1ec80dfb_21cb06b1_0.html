字符串

    可使用单引号或双引号

    字符串必须总是被写在一行中（??）

    

在字符串中使用引号
       
    在双引号内使用单引号，或反之
 
      
    可以使用转符: \



字符串操作符
       
    字符串连接：+
       
    数字和字符串连接：+
       
    对字符串使用算术操作符，得到 NaN（not a number）



示例
       
    使用 innerHTML 属性，添加 HTML 内容到页面上。
               
        message = '<a href=\"sale.html\">25% off!</a>';
        var elNote = document.getElementById('note');
        elNote.innerHTML = message;
