git add - 向暂存区中添加文件

    工作树中的文件不会被记入 Git 仓库的版本管理对象中，用 git status 查看会显示在 Untracked files 里
    
    让文件成为 Git 仓库的管理对象，需要用 git add 将其加入暂存区（Stage 或 Index)中

    暂存区是提交之前的一个临时区域


    ## 将 README.md 加入暂存区
    $ git add README.md
    
    ## 加入暂存区后，利用 git status 查看，README.md 文件显示在 Changes to be commited 中
    $ git status

    ## 将工作树中的所有内容加入暂存区
    $ git add .
