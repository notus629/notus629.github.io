Rewrite Flags

    The behavior of a RewriteRule can be modified by the application of one or more flags to the end of the rule.

        ##  the matching behavior of a rule can be made case-insensitive by the application of the [NC]
        RewriteRule "^puppy.html" "smalldog.html" [NC]


    [F] flag     causes the server to return a 403 Forbidden status code to the client
