Substitution

    Can be one of the three things


    1. A full filesystem path to a resource

        RewriteRule "^/games" "/usr/local/games/web"


    2. A web-path to a resource

        ## If DocumentRoot is set to /usr/local/apache2/htdocs, then this directive would map requests for
        ##    http://example.com/foo to the path /usr/local/apache2/htdocs/bar.
        RewriteRule "^/foo$" "/bar"

    
    3. An absolute URL

        ## This tells the client to make a new request for the specified URL.
        RewriteRule "^/product/view$" "http://site2.example.com/seeproduct.html" [R]

        ## Substitution can also contain back-references
        ## request for http://example.com/product/r14df/view will be mapped to /var/web/productdb/r14df
        RewriteRule "^/product/(.*)/view$" "/var/web/productdb/$1"
