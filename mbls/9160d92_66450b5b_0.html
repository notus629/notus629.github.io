1. 调用外部JS文件
        <script src=".../*.js"></script>
        // 可放在head中，也可以放在body的任意元素中，在放置位置处产生输出。

2. 直接在html中写入js代码

       <script type="text/javascript">        
           // 若为JavaScript文件 text/javascript可省
           // js codes here
       </script>



3. 使用language属性（在W3C的HTML标准中，已不再推荐使用）
<script language="Javascript"></script>
