SDK 解读 -- SDK 核心类和重要方法（saetv2.ex.class.php 文件）

    OAuthException 为空，可以删除


    SaeTOAuthV2

        跟授权相关的类，获取 code, 再由 code 得到 accesstoken 和 refreshtoken, 等和开放式授权相关的功能。


    SaeTClientV2

        如何去调用微博的API（发布微博、已发布的微博等接口）



SaeTOAuthV2

    __construct($client_id, $client_secret, $access_token = NULL, $refresh_token = NULL)

        client_id: app id

        client_secret: app secret


    authorizeURL(); 

        返回微博登陆授权页面的主 URL, 在此链接后拼接相关参数，用户登录授权后返回 code


    getAuthorizeURL(...);

        拼接相关参数到登录授权页，跳转访问微博登录页。

        用户登录授权后，跳转到回调地址


    accessTokenURL();  

        拿到 code 之后，将 code 拼接到此链接后面，拿到 accesstoken  


    getAccessToken(...);

        拼接 code 到 accesstoken 请求地址，通过 post 来请求 accesstoken 和 refreshtoken

   
    拿到 accesstoken 后，就可以根据 SaeTClientV2 类来调用具体的接口。



SaeTClientV2

    __construct( $akey, $skey, $access_token, $refresh_token = NULL)

        refresh_token  一般不需要，除非需要执行一些定时任务
