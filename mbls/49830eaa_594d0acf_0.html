clone 已有仓库



clone 地址

    地址格式为： git@github.com:github_username/repository_name.git

    可在项目页面：Clone or download -> Clone with SSH 查看



clone 项目到本地

    ## 会将项目名（GitHub 上的 Repository name）作为文件夹，里面包含 git 仓库(即 .git 隐藏文件夹）和项目文件
    ## 下载到 git bash 当前目录
    ## clone 前提示输入 ssh key 的密码
    $ git clone git@github.com:github_username/repository_name.git
