CenOS 5.11 下快速搭建 LAMP - 使用yum命令

       Leo, 2016-11-14，VirtualBox虚拟机，CentOS 5.11


    1. 确保虚拟机可以连网, 这时使用 yum 安装, 它可以帮助我们解决软件自己的依赖关系，安装命令如下

        $ yum -y install httpd mysql mysql-server php php-mysql php-devel


    2. 启动 apache 服务并查看是否启动成功, 命令如下:（切记用 root 用户启动服务）

        ## 启动
        $ /etc/rc.d/init.d/httpd start

        ## 检测启动结果
        $ ps aux | grep httpd


    3. 查看服务器的 ip 并进入网站根目录新建一个 phpinfo.php 的程序，用于查看 php 相关配置信息。

        $ ifconfig
        $ cd  /var/www/html/
        $ cat phpinfo.php
        <?php
            echo phpinfo();

         
    4. 进行 mysql 数据库的配置

        启动：  $ /etc/rc.d/init.d/mysqld start

        检查启动结果：  $ netstat -tulnp | grep :3306

        修改 root 密码： $ mysqladmin -u root password '567890'       


    5.  连接数据库，并进行测试

        # mysql -uroot -h localhost -p567890
