两种方式安装 Laravel

    通过 $ composer create-project 命令安装 Laravel


    通过 composer 安装 Laravel 安装器，再通过 Laravel 安装器来安装 Laravel 框架

        $ composer global require "laravel/installer"

        $ laravel new blog    ## 项目名称叫 blog



通过 composer create-project 命令安装

    ## blog 为项目的名字，可以自定义
    $ composer create-project --prefer-dist laravel/laravel blog



通过 Laravel 安装器安装

    a. 首先下载安装器

        $ composer global require "laravel/installer"


    b. 安装完成后，将 $HOME/.composer/vendor/bin (即 laravel 可执行文件路径) 添加到 PATH 环境变量中

        Mac/Linux: vim ~/.bash_profile, 加入 export PATH=~/.composer/vendor/bin:$PATH

        Windows: C:\Users\Gene\AppData\Roaming\Composer\vendor\bin

    
    c. 再安装 laravel (blog 为项目名称)

        $ laravel new blog  
    

 
下载 Laravel 最新的开发版 (dev 版)

    $ laravel new blog --dev
