系统管理

命令说明
certmgr.msc证书管理实用程序
cliconfgSQL SERVER 客户端网络实用工具
compmgmt.msc计算机管理
dcomcnfg打开系统组件服务
eventvwr事件查看器
gpedit.msc本地组策略编辑器
lusrmgr.msc本地用户和组（本地）
mmc打开 Microsoft 管理控制台（Microsoft Management Console, mmc）
odbcad32ODBC数据源管理程序（64位）
perfmon 或 perfmon.msc性能监视器
perfmon /res资源监视器
rsop.msc组策略结果集
secpol.msc本地安全策略
services.msc本地服务设置 （打开“服务”管理工具）
syskey保证 Windows 账户数据库的安全（系统加密，一旦加密就不能解开）
wmimgmt.msc打开 Windows 管理体系结构(WMI)
wscriptWindows 脚本宿主设置(仅设置图标和终止脚本时间)
