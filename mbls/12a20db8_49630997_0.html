win10 cmd 调整

        透明效果：ctrl + shift + '+/-'
        全屏：alt + enter
        F7，然后通过菜单查看之前使用过的命令，回车即可再次执行。

        支持功能：
        
                拖拽文件夹快速输入路径
                预测补全： tab键
                ctrl + C/V
                文本搜索：ctrl + F


Window 上常用的命令

appwiz.cpl 
打开卸载或更改程序
control
打开控制面板
mstsc打开远程连接
mspaint
打开画图程序
ncpa.cpl 
打开网络连接
netstat -ano
查看端口被占用情况
netstat -ano | findstr "8080"
同上并过滤8080关键字
netstop http
停止某项（如http)服务
notepad                    打开记事本
services.msc
打开"服务"快捷命令
