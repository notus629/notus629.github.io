markdown

    PHPStorm 2016.2.2 已经自带 markdown 插件功能

    可以在 Settings -> Languages & Frameworks -> Markdown 中进行设置

    自带语法高亮和预览功能



gfm [1]

    使用 GitHub flavor markdown 语法的插件

    It integrates as a tab at the bottom of markdown documents
