初始设置



首先设置姓名和邮箱地址

    $ git config --global user.name "Gene Leo"

    $ git config --global user.email "geneleo@qq.com"


    亦可直接编辑 ~/.gitconfig 进行设置

        [user]

            name = Firstname Lastname

            email = your_email@example.com


    这里设置的姓名和邮箱地址会用在 Git 的提交日志中



提高命令输出的可读性

    将 color.ui 设置为 auto

        $ git config --global color.ui auto


    在 "~/.gitconfig" 中会增加下面一行

        [color]

            ui = auto
