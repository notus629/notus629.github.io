CentOS 7 安装 Redis

    1. 下载解压，同 linux 安装步骤。

        ## or download http://download.redis.io/releases/redis-stable.tar.gz
        $ wget http://download.redis.io/releases/redis-3.2.8.tar.gz
        $ tar xzvf redis-3.2.8.tar.gz


    2. 检查 gcc 是否安装，否则须先安装 gcc, 否则 make 时会出现 cc 命令找不到的错误。 

        $ yum list gcc
        $ yum install gcc 
   
        若再出现"cc 命令找不到"错误后再安装 gcc, 则再次运行 make, 会出现 “jemalloc.h no such file or directory”，可运行 $ make distclean 进行清除 [3]


    3. 编译，运行 $ make


    4. 根据提示，运行 $ make test, 提示 "you need tcl to run the redis test", 安装 tcl. 安装之后, 再次运行 $ make test 会成功

       $ yum install tcl
