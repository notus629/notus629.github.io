Ajax

    Asynchronous JavaScript And XML (异步 JavaScript 和 XML），表示一组在浏览器中提供异步功能的技术。

    无须刷新整个页面就能为页面中某一部分加载数据的技术

    数据通常会以 JavaScript 对象表示法 （JavaScript Object Notation, JSON) 的格式来进行发送

    Ajax 使用异步（非阻塞）处理模型



服务器返回格式

    HTML, XML, JSON


    浏览器仅允许 Ajax 加载来自和页面其余部分相同域名的 HTML 和 XML

        如：页面位于 www.example.com ，Ajax 必须从 www.example.com 返回数据
