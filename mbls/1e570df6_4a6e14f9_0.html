变动事件和变动观察者



变动事件

    任何时候当元素被添加到 DOM 中或从 DOM 中移除时，DOM 的结构就发生了变化，这种变化会触发变动事件

    变动事件已经计划被变动观察者取代


事件触发
DOMNodeInserted当一个节点被插入到 DOM 树中时触发，如使用 appendChild(), replaceChild() 或 insertBefore() 时
DOMNodeRemoved当一个节点从 DOM 树中被移除时触发，如使用 removeChild 或 replaceChild() 时
DOMSubtreeModified当 DOM 结构发生变化时触发，它会在上述两个事件发生后再触发。
DOMNodeInsertedIntoDocument一个节点被插入为另一个节点的后代。当一个节点作为后代节点被插入到另一个已经存在于文档中的节点时触发。[2]
DOMNodeRemovedFromDocument一个节点被从其祖先节点上移除

  

变动观察者

    变动事件在脚本大量地修改了页面时，可能会造成页面速度变慢甚至失去响应。

    变动观察者，在进行响应之前等待脚本做完所有工作，然后再将 DOM 的变化批量地报告给脚本（而不是每次都报告）
