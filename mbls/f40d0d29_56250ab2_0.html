数字类型

        整数数据类型
                TINYINT[M] / BIT / BOOL
                SMALLINT[M]
                MEDIUMINT[M]
                INT[M] / INTEGER[M]
                BIGINT[M]

        浮点数据类型
                FLOAT(精度)
                FLOAT[M,D]
                DOUBLE[M,D] / PRECISION[M, D] / REAL[M, D]
                DECIMAL / NUMERIC / DEC / FIXED[M,D]

        整型数据后： UNSINGED, 指定为无符号型
        所有数字类型：ZEROFILL 属性，显示该字段中的值时，空余部分用前导0来补充，整型指定为ZEROFILL，则自动unsigned
