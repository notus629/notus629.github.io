字体



字体属性

    font                 一条声明中设置字体的简写属性

    font-family          指定文本块采用的字体名称    

    font-size            指定文本块的字体大小

    font-style           指定字体样式

    font-variant         指定字体是否以小型大写字母显示

    font-weight          设置字体粗细




字体相关 - 示例

    font: bold 24pt sans;                  // 字体设置

    font-size: 14px;                       // 字体大小

    line-height:1.75;                      // 

    padding: 50px;

    background-color: #f5f5f5;
    padding: 10px 15px;
    border: 1px solid #ccc;
    border-radius: 5px;
