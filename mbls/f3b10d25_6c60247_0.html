PuTTY SSH 连接一段时间就段开

    原因是在于 SSHD 的设置问题

    解决方案 1：

        修改/etc/ssh/sshd_config配置文件，设置ClientAliveCountMax值大一点，单位是分钟。

        然后重启 ssh 服务使生效：service sshd reload

    解决方案 2：

        putty -> Connection -> Seconds between keepalives ( 0 to turn off ), 默认为0, 改为300. （即每隔300秒向服务器发送一个keepalive）
