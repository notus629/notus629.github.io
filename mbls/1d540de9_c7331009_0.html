网页中启用 QQ 交谈

    1. 打开 [2]，提示登录QQ。

    2. 选择三种组件样式之一：简洁型，标准型，拓展型

    3. 设置提示语，默认为：“点击这里给我发消息”

    4. 将下方生成的代码粘贴到 html 代码中即可。如

1
2
3
<a target="_blank" href="http://wpa.qq.com/msgrd?v=3&uin=6419574&site=qq&menu=yes">
    <img border="0" src="http://wpa.qq.com/pa?p=2:6419574:51" alt="点击这里给我发消息" title="点击这里给我发消息"/>
</a>



注：

    生成 “QQ 交谈代码” 的 QQ，其 “QQ在线状态” 服务须先启用
