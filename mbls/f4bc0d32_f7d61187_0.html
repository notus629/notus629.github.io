Ubuntu 源列表更新

    方法1：打开 Ubuntu 的 "software center"->"Edit" 菜单->"Software Sources"->other->select Best server 会自动测速


    方法2：修改源列表 

        在添加前建议手动 ping 来验证

        首先备份源列表

            $ sudo cp /etc/apt/sources.list /etc/apt/sources.list_backup 

        而后用gedit或其他编辑器打开

            $ gedit /etc/apt/sources.list

        从列表（网上查找的）中选择合适的源，替换掉文件中的所有内容，保存编辑好的文件

        然后刷新列表

            $ suto apt-get update



更新源时，Ubuntu 更新源出现 Hash Sum mismatch 错误

    这是由于GFW的原因，我们获取到的文件和官方源上的文件大小不一致，导致md5值也不一样，所以会出现Hash Sum mismatch。我们可以通过设置代理的方式解决这个问题。
