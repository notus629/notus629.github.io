将项目公开到GitHub仓库

    已在 GitHub 上创建了一个新仓库

    项目存放在本地，需要将其添加到 GitHub 上的新建仓库



若本地项目还未进行 Git 管理 （无本地仓库，未提交到本地仓库）

    步骤：

        1. clone GitHub 已有仓库到本地

        2. 本地项目代码添加到本地仓库 (将项目内容拷贝至 clone 的文件中)

        3. 将本地仓库 push 到 GitHub 仓库中



若本地已有仓库（已将项目提交到本地仓库）

    若 GitHub 上新建仓库为空（不包含 README.md, .gitignore, LICENSE)

        git clone 不会冲突

        git clone 后再 push 到 GitHub 仓库中即可


    否则将 GitHub 上新仓库 git clone 到本地时，会提示冲突（GitHub 上新建仓库包含额外文件：README.MD, .gitignore, LICENSE)

        解决冲突后，再依照上面的步骤处理
