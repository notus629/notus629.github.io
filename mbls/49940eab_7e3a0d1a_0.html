本地项目提交



本地项目添加到 git 项目目录中

    先将项目拷贝到本地 git 项目文件夹（即 clone 到本地的 repository_name/ 目录）下
    
    可以相应的添加并修改：.gitignore, README.md 和项目源代码



本地项目提交到本地仓库

    ## 查看本地工作区状态，哪些文件未提交
    $ git status

    ## 添加所有工作区文件到暂存区
    $ git add .

    ## 提交暂存区中的文件到本地仓库，并添加本次提交说明
    $ git commit -m "first commit for 'message board based on vue.js'"

    ## 添加成功后，通过 git log 命令查看提交日志
    $ git log
