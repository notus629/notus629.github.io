cURL 的使用场景

    网页资源

        - 编写网页爬虫


    WebService 数据接口资源

        - 获取动态的接口数据，比如天气、号码归属地等等


    FTP 服务器里面的文件资源

        - 下载 FTP 服务器里面的文件


    其他资源

        - 所有网络上的资源都可以用 cURL 访问和下载到



在 PHP 中使用 cURL

    确认 PHP 支持 cURL (或直接看 phpinfo() )

        Windows: cmd > php -i  ## 打印出 PHP 所有当前设置，查找 cURL
   
        Linux: $ php -i | grep cURL
