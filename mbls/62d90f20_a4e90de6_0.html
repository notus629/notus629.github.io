用户界面(UI)事件

    UI 事件的事件处理程序/监听器附加在浏览器窗口上


    load        当 Web 页面加载完成时触发，同样可以用于其它元素节点的加载事件，如图片、脚本或对象。
                在第2级 DOM 规范中规定它应当在 document 对象上触发，一般浏览器也支持旧的 window 对象上触发。（实际 window 用的更多）

    unload      当 Web 页面卸载时（通常是因为请求了一个新页面）发生，该事件会在用户离开页面之前触发。
                在第2级 DOM 规范中规定它应当在 <body> 元素节点上触发，一般浏览器也支持旧的 window 对象上触发。   

    error       当浏览器遇到 JavaScript 错误或不存在的资源时触发。
                不同的浏览器对此事件的支持不一致。

    resize      当浏览器窗口改变大小时触发。调整浏览器窗口大小的过程中，此事件会不停的触发。

    scroll      当用户向上或向下滚动页面时触发。可以关联整个页面，也可以关联页面中的某些特定元素（如有滚动条的 textarea)
                调整浏览器窗口大小的过程中，此事件会不停的触发。
