Git 本地仓库操作基本命令

    1. 若第一次使用，则需要先进行用户信息配置（全局配置）

        $ git config --global user.name leo
        $ git config --global user.email hehe_xiao@qq.com


    2. 接着对某个项目进行版本管理，进入项目所在目录，进行 git 版本库初始化

        ## 会在当前目录下创建隐藏的 .git 目录，用于 git 跟踪版本管理
        $ git init


    3. 将修改过的文件由工作区添加到暂存区（每次修改须重新添加）

        ## 添加单个文件
        $ git add some.php
        ## 添加所有文件
        $ git add .

    
    4. 查看文件状态（是否有文件未提交）

        # git status


    5. 把文件提交到仓库(本地仓库)

        ## 若未带 -m 选项，会打开一个 vim, 填写说明并保存退出之后才会真正提交
        $ git commit -m "本次提交的说明"


    6. 查看已提交信息

        $ git log
