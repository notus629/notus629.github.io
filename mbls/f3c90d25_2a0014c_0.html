Starting MySQL Server as a Microsoft Windows Service:
       mysqladmin -p -u root shutdown
       mysqld install
       Once a MySQL server has been installed as a service, Windows starts the service automatically whenever Windows starts. The service also can be started immediately from the Services utility, or by using a NET START MySQL command.
