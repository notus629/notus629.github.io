安装



Mac & Linux

    Mac 已预装 Git

    Linux 以软件包（Package)的形式提供给用户了



Windows 下安装

    选择 msysGit 安装包 [1]

    设置调用 Git 的环境变量，选择 Git Bash 命令提示符。


    换行符处理 

        GitHub 中公开代码，大部分以 Linux 中的 LF(Line Feed，即 \n)换行, Mac(\r), Windows(CRLF, \r\n)

        选择 "Checkout winodws-style, commit Unix-style line endings", 则签出时（check out)自动转换为 CRLF, 提交时（commit）自动转换为 LF.        
    

    Git Bash, 还包含了很多 Bash 命令
