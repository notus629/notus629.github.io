镜像说明

    安装包的数据是从 github.com 上下载的，而安装包的元数据是从 packagist.org 上下载的

    国外网站连接速度慢，并随时可能被“墙”

    Composer 中国镜像，缓存所有元数据和安装包到国内的机房，并通过国内的 CDN 进行加速



镜像配置 [1]

    查看当前的镜像地址（Win/Mac/Linux）

        ## notus 注：无法查看，改用 composer config -gl
        $ composer config -g repo.packagist

        单个项目配置



单个项目镜像配置

    将配置信息添加到某个项目的 composer.json 文件中

    到项目根目录，执行以下命令
        $ composer config repo.packagist composer https://packagist.phpcomposer.com



系统全局镜像配置

    $ composer config -g repo.packagist composer https://packagist.phpcomposer.com
