重启网卡

    $ service network restart                        ## 重启所有网卡
    $ service network start | stop                   ## 打开/关闭所有网卡
        
    ## or
    $ ifconfig eth0 up | down                        ## 打开、关闭特定的网卡(eth0)



查看是否启用了 sshd (open ssh 服务)

    $ ps -ef | grep sshd                             ## 查看进程中是否有 sshd 服务 

    $ /etc/init.d/sshd  status                       ## 查看 sshd 服务是否运行



启动 sshd 服务 

    $ /etc/init.d/sshd start



查看端口状态

    $ netstat -anp  | grep ssh                       ## 哪些端口被打开



关闭某些端口（禁用端口）      





关闭对应的应用（即关闭端口，但未禁用，别的应用还可继续使用）

    $ kill  -9   <PID>   
 


查看发行版版本号

    $ cat /etc/issue

    ## 更详细的信息, Ubuntu16.04 有此命令，而cenos7没有（需安装，不推荐）
    $ sudo lsb_release -a

    ## centos使用 cat /etc/redhat-release


查看端口占用情况

    $ netstat -ntlp



查看环境变量，变换行显示

    $ echo -e ${PATH//:/"\n"}
    

压缩解压

    ## zip 文件解压到当前目录
    $ unzip filaname.zip



查看磁盘空间

    $ df -h
