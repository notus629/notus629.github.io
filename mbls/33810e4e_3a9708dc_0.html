基本步骤



处理 Ajax 请求及响应

    浏览器实现了一个 XMLHttpRequest 的对象来处理 Ajax 请求。


    基本步骤：

        1. 浏览器使用 XMLHttpRequest 对象来创建 Ajax 请求

        2. 服务器响应浏览器的请求

        3. 同一个 XMLHttpRequest 对象继续处理返回的结果



1 . Ajax 请求

    // 创建 XMLHttpRequest 对象
    var xhr = new XMLHttpRequest();
    // 准备请求，参数 1: HTTP 方法； 参数 2：处理请求的页面地址； 参数 3： 是否异步
    xhr.open('GET', 'data/test.json', true);
    // 将准备好的请求发送给服务器。参数：包含的额外信息，若无额外信息，放空或使用 null 
    xhr.send('search=arduino');



3. 处理服务器发回的响应

    // 浏览器收到来自服务器的响应并将其载入时，会触发 onload 事件
    xhr.onload = function() {
        // 确保服务器的请求是正常的
        if(xhr.status == 200){
            // code to process the results from the server
        }
    }
