定时任务命令

    1. 定时任务提供 crontab 命令来设定服务

    2. crontab -e   // edit, 编辑某个用户的 cron 服务

        */1 * * * * /usr/bin/php /data/www/12.php

    3. crontab -l   // list, 列出某个 cron 服务的详细内容

    4. crontab -r   // 删除某个用户的 cron 服务



定时任务 crontab 格式

    分    时    日    月    周    命令
    *     *     *     *     *    
   0-59  0-23  1-31  1-12  0-6   command



定时任务  crontab 示例

    */1 * * * * php /data/www/cron.php   每分钟执行 cron.php
    
    50 7 * * * /sbin/service sshd start  每天 7:50 开启 sshd 服务
