事件对象

    每个事件处理函数都会获得一个事件对象，在小括号中以命名参数的方式出现（通常为 e 或 event)

        $('li').on('click', function(e){
            eventType = e.type;
        });


属性说明
type事件类型（如 click, mouseover)
which被按下的按钮或键
data事件发生时，传入该函数的一些额外信息
target发生了这个事件的 DOM 元素
pageX鼠标相对于可视区域左边缘的位置
pageY鼠标相对于可视区域右边缘的位置
timeStamp事件发生时间距离 1970 年 1 月 1 日的毫秒数（即 Unix 时间）。Firefox 中无效


方法说明
.preventDefault()阻止默认行为（如提交表单等）
.stopPropagation停止事件向其祖先元素中冒泡传播

    <script type="text/javascript" src="./js/jquery-3.1.1.min.js"></script>
    <script type="text/javascript">
        $(function(){
            $('li').on('click', function(e){
                $('li>span').remove();
 
                // 创建新的日期对象, 并将其时间设置为点击事件发生的时间
                var date = new Date();
                // timeStamp 在 chrome (Version 58.0.3029.81 (64-bit))中为事件发生时刻距页面生成时间的毫秒数，firefox （2017-4-27）计算不正确
                date.setTime(e.timeStamp);
                // 将事件发生的时间转换为可读的日期
                var clickDate = date.toDateString();
 
                $(this).append('<span> ' + clickDate + ' ' + e.type + ' </span>');
 
            });
 
        });
    </script>
