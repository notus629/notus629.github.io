composer

    是 PHP 的一个依赖（dependency) 管理工具，不是一个包管理器, 它涉及 packages 和 libraries

    在项目中声明所依赖的外部工具库（libraries), Composer 会自动安装这些工具库及依赖的库文件。



保持最新版

    执行 $ composer selfupdate 以保持 Composer 一直是最新版本
