居中解决方案



利用弹性盒子 [3]

    容器: display: flex;

    元素: margin: auto;



水平居中

    他们起作用的首要条件是子元素必须没有被 float 影响


    text-align: center;        // 内容居中，针对子元素

    margin: 0 auto;            // 元素本身居中，针对父元素， 注意 position 不能是 absolute;

    margin: 0 auto 0 auto;     // 同上， 注意 position 不能是 absolute;
