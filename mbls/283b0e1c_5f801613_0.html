CentOS 上搭建一个远程仓库 - 步骤如下


    1. 安装 Git

        # yum install -y git                 ## 安装
        # git --version                      ## 查看 git 版本

    
    2. 创建 git 用户来管理 git 服务，并为 git 服务设置密码

        # id git                             ## 判断系统中是否已存在 git 用户
        # user add git                       ## 若系统中无 git 用户，则创建名为 git 的用户
        # passwd git                         ## 根据提示来修改 git 用户的密码


    3. 服务器端创建 Git 仓库

        # git init --bare sample.git
        $ sudo chown -R git:git sample.git
