访问元素节点

    DOM 查询可能返回一个元素

    也可能返回一个 NodeList, 即元素的节点的集合



选择单个元素 - 返回单一节点

    document.getElementById('one')                  // 根据 ID

    document.querySelector('css selector')          // 使用 CSS 选择器，返回第一个匹配的元素



返回一个或多个元素节点（返回 NodeList)

    document.getElementsByClassName('class')               // class 属性中使用了特定值的元素

    document.getElementsByTagName('tagName')               // 使用了指定标记的元素（即元素标签）

    document.querySelectorAll('css selector')              // 使用 CSS 选择器来选择所有匹配的元素



在元素节点间遍历（以下皆为属性而非方法）

    parentNode

    previousSibling/nextSibling

    firstChild/lastChild

    若是对应的元素节点不存在，则会返回 null
