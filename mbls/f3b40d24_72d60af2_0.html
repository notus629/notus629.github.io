常用命令

    $ memcached -h           ## 可查看相应的使用帮助

    $ memcached -d -uroot    ## 以 daemon 方式启动

        要再加上-uroot, 否则会出现 "can't run as root without the -u switch" 的错误。



$$$$ 注 $$$$

    memcached 不能以 $service memcached start 方式启动，因为在安装时，并没有在 /etc/init.d/ 并不存在相应的服务启动脚本，因而也不能用 chkconfig 来设置自启动。

    可以自已编写相应的启动脚本，具体参见 [1]
