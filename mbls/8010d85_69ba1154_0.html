以 shell 方式启动 memcached

    ## 关闭控制台则退出
    > memcached.exe -p 端口号


    
停止 memcached

    > memcached.exe -d stop

telnet 操作

    1. 登录到 telnet 连接到 memcached 服务

        telnet 127.0.0.1 11211


    2. 增加

        add key名 0(是否压缩) 存放时间(s) 数据大小(字符)

        add key1 0 30 5, 回车，再输入"hello"


    3. 获取值

        get key1


    4. 修改
    
        set key名 0 存放时间 数据大小

        replace key名 0 存放时间 数据大小

    
    5. 删除

        delete key名 


    6. 其它指令

        append, prepend, incr, decr, flush_all, flush_all 900, stats
