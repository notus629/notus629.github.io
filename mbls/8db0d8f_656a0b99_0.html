引入 html 页面外部的 css 代码

    <!-- 需要在 head 元素中引入 -->
    <!-- 若是 css 的话，type 属性通常可省 -->
    <link rel="stylesheet" type="text/css" href=" *.css" />
   


在 html 内部嵌入 css

   <!-- 需要在 head 元素中引入 -->
   <style>
       <!-- css codes -->
   </style>



元素内嵌 - 作为其它 HTML 元素(如 font)的 style 属性

   <font style="color: red;">...</font>
