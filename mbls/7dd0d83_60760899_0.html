Memcache Module in PHP [1]

    Memcache module provides handy procedural and object oriented interface to memcached, highly effective caching daemon, which was especially designed to decrease database load in dynamic web applications.

    The Memcache module also provides a session handler (memcache).



依赖库 zlib [5]
 
    This module uses functions of zlib to support on-the-fly data compression. Zlib is required to install this module.



memcache 安装 - Windows [2]

    下载 memcache 对应 PHP 版本（TS/NTS, X86/X64, php version no.)的动态库，放入相应文件夹，再修改 php.ini 即可



memcache 安装 - Linux [3]

    CentOS 7: $ yum install php-pecl-memcache

    再在 php.ini 中添加动态库

    再重启 nginx 和 php-fpm



memcache 配置 (Runtime Configuration) 

    可在 php.ini 里进行配置，详细功能参见 [4]



memcache 的资源类型

    There is only one resource type used in memcache module - it's the link identifier for a cache server connection.
