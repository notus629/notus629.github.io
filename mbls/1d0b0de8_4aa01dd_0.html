常用快捷键：

        ctrl + shift + /                                     // 块注释
        Ctrl + Q                                             // (View | Quick Documentation)
        选中左侧的文件名  shift + f6(mac需要同时按住键)         // 文件重命名
        ctrl + b, or click mouse & ctrl                      jump to the declarations: 

        ctrl + J                                             写一段文字之后，按ctrl+j自动补全
        alt + 1                                              查看文件列表
        alt + 7                                              查看类的函数列表
        ctrl + p                                             显示函数参数提示



windows 下 phpstorm 的快捷键

    ctrl+shift+n                    查找文件
    CTRL+ALT ←/→                    返回上次编辑的位置(与windows 的屏幕横向纵向快捷键冲突)
    ALT+ ←/→                        切换代码视图，标签切换
    ALT+ ↑/↓                        在方法间快速移动定位
    ctrl+shift+f                    在一个目录里查找一段代码(ctrl+f的升级版)
    ctr+shift+r                     在一个目录里查找一段代码并替换(ctrl+r的升级版)


       ctrl+d复制当前行

       ctrl+shift+u字母大小写转换

       ctrl+r替换
       F4查看源码
