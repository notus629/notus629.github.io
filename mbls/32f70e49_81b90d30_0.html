GitHub 上创建仓库

    在 GitHub 菜单上，可以 New repository, 创建全新仓库


    或 Import repository, 从旧仓库的克隆地址 (clone URL) 导入（支持 svn 等 VCS）

        Import all the files, including the revision history, from another version control system.

        只能从其它 version control system 地址导入，并不能从本地上传



Repository name

    仓库名

    本仓库的地址即为：https://github.com/github_username/repository_name



Public or Private

    创建时可选择 Public 或 Private, 选择 Private 需要对账户进行升级



Initialize this repository with a README

    选项： "Initialize this repository with a README", 会自动初始化仓库并设置 README 文件（README.md）

    若以手动方式添加已有的 Git 仓库，不勾选此项



Add .gitignore

    记录不需要在 Git 仓库中进行版本管理的文件, 初始化时自动生成

    默认为 None，即不创建

    下拉列表中有各种框架，即会根据框架进行自动设置不需要版本管理的文件

    一般没有框架选择不创建 .gitignore 文件，有框架也可选择不创建



Add a license

    选择要添加的许可证协议文件，目前用的比较多的是 MIT 协议

    一般还会在 README.md 文件中说明声明项目所采用的协议。（可参见知名开源项目的 README.md 文件）
