    PECL [1]

        The PHP Extension Community Library（PECL，pronounced "pickle"）， is a separate project that distributes PHP extensions。 (compiled code written in C, such as the PDO extension, Windows dlls).

        PECL extensions are also distributed as packages and can be installed using the PEAR installer with the pecl command.


    PEAR

        官方维护的软件包集合，PEAR核心包随着PHP绑定发布。

        PEAR is short for "PHP Extension and Application Repository"。The purpose of PEAR is to provide:

            * A structured library of open-source code for PHP users
            * A system for code distribution and package maintenance
            * A standard style for code written in PHP
            * The PHP Extension Community Library (PECL)
            * A web site, mailing lists and download mirrors to support the PHP/PEAR community

        pear命令，当前目录，c：\php下
            go-pear                                安装pear
            pear list-all                          列出所有可供安装的包的名称
            pear list                              查看当前系统已经安装的PEAR包
            pear install Mail_Mime                 安装MIME邮件包，安装其它包类似
            pear upgrade pkgname                   查看已经安装的包的最新版本
