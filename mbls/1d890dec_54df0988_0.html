引入官方 SDK

    即官方登录的示例代码，可以根据需要自行修改



目录结构

    API/     接入 QQ 登录的核心类，配置文件，项目中使用到的就此一个

    DOC/     帮助文档

    example/ 接入QQ互联以后，用API开发的一些功能的示例

    install/ 帮助配置 SDK 参数的小工具，配置完之后此文件夹就可删除



SDK 参数配置

    访问 SDK 所在目录，会自动进行 install

    配置项：

        请求授权列表 (根据需要选择，最好不要全选上）
            get_user_info    获取用户基本信息



SDK 解读

     文档资料 -> oauth 开发指引 -> 开发攻略_Server-side

     Server-side or Client-side



SDK 核心类和重要方法

    登录授权相关的三个主要类 (Connect2.1/calss/*.class.php)
      
        Recorder.class.php [配置读写与 SESSION 存取]
     
        URL.class.php [基于 CURL 库的 get 与 post 请求]           
        Oauth.class.php [Oauth 相关 URL 动态拼接与 token 操作]
