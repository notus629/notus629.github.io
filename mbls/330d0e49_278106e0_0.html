buffer 概述

    buffer 其实就是缓冲区， 一个内存地址空间，主要用于存储数据区域



输出流程

    输出内容 -> php buffer -> tcp -> 终端



开启 output_buffering

    php.ini 文件中
        output_buffering = on          ;; 默认开启 output buffer 

    或 ob_start() 函数



ob_get_contents() 函数

    得到缓冲区中的内容，output_buffering 须开启，如

        ob_get_contents();
