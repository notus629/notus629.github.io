会话相关函数

    session_start();                         // 开启session支持

    session_get_cookie_params()              // 查看由会话控制设置的cookie内容

    session_set_cookie_params()              // 设置会话 cookie 的参数

    session_destroy();                       // 清除一个会话ID

    session_unregister();                    // 不推荐使用

    session_unset();                         // 不推荐使用 



概念

    PHP的会话是通过唯一的会话ID来驱动的

    会话 ID 是一个加密的随机数字，由PHP生成，在会话的生命周期都会保存在客户端。

    它可以保存在用户机器的 cookie 中，或通过 URL 在网络在传递。



存储会话会话 ID

    默认，PHP 将在会话中使用 cookie，可以设置一个 cookie 来存会话 ID

    可以将会话 ID 添加到 URL 中，通过在 php.ini 文件设置 session.use_trans_sid 来实现

    会话 ID 保存在常量 SID 中，手动通过链接传递会话 ID
        <a href="link.php"?<?php echo strip_tags(SID); ?>">
