不同 PHP 下载类型的选择 [1]

    IIS

        如果你在使用 IIS 把 PHP 作为 FastCGI，则选用 PHP 的 NTS （Non-Thread Safe)版本。（多进程，由 fastcgi-fpm 管理，所以用 NTS， Leo 注）


    Apache

        首先要使用正确的 Apache 版本（由 Apache Lounge 提供）（主要是 VC9+ 以上的版本）, PHP 的 SAPIs 即由对应的版本来生成的。

        而以 apache.org 提供的版本（不建议），来使用 PHP 模块的话，只能使用老的 VC6.

        Apache 下，使用 TS （Thread Safe) 版本. (此时，PHP 以模块的方式，以多线程来处理。Leo 注）



不同的 VC 版本 [1]

    VC9 版本，需要安装 VC++ 2008 SP1

    VC11 版本，需要安装 VC++ 2012 

    VC14 版本，需要安装 VC++ 2015



TS 和 NTS [1]

    TS 

        指可以进行多线程处理的版本

        TS 版本可以与多线程的 SAPI 进行交互，PHP 以模块的形式被载入 Web 服务器。        

    NTS 

        指单线程版本
        
        最常见的形式是通过 FastCGI 协议和 Web 服务器进行交互, 不使用多线程。

        CLI 也是以这种方式处理。
