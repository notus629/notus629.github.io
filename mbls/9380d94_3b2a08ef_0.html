对象概念

    名称和值

       
    对象中的名称：键
       

    值

        属性的值：字符串、数字、布尔值、数组、另一个对象
               
        方法的值：只能是函数

            var hotel = {
                name: 'Quay',
                rooms: 40,
                booked: 25,
                roomTypes: ['twin', 'double'],
                checkAvailability: function() {
                    return this.rooms - this.booked;
                }
            };
