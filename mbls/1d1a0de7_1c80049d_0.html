Init Scripts

    对于 CentOS 7，yum 安装，是 /etc/sysconfig/memcached 文件，默认配置如下：

        PORT="11211"
        USER="memcached"
        MAXCONN="1024"
        CACHESIZE="64"
        OPTION=""
    

    对于自己生成(building)的 memcached, 源码包的 scripts/ 目录下有很多 init scripts 可供参考。    

    改变端口号后，即可运行多个 memcached 实例，如 memcached -p 11212
