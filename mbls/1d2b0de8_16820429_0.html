Networking

    By default memcached listens on TCP and UDP ports, both 11211.    

    -l, 绑定到特定的接口或 IP 地址。
    
    -p, 改变将监听的 TCP 连接端口，以这种方式改变端口会使得 UDP 端口跟着变化

    -U, 修改 UDP 端口，set 0 to disable it.



Unix Sockets

    若只想在本地的单个用户使用，不暴露在网络中，可以使用 unix domain socket

    -s <file>, 开启之后，TCP/UDP 被禁用
