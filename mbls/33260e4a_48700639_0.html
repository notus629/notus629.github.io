Win10+PHP7.0+Apache2.4

    搭配 apache 2.4 服务器

    下载 TS (thread safe) 版(因为apache要以sapi模块的多线程方式运行)， 下载文件为 php-7.0.13-Win32-VC14-x86.zip



安装步骤

    1. 暂停服务器（Apache, 或 Nginx)


    2. 解压，并命名为php文件夹，存到c:/php（之前的c:/php改为其它名字或移除）


    3. 将php.ini-development改名为php.ini

    
    4. 修改 php.ini 文件

        extension_dir 修改为扩展DLL目录的位置，即extension_dir="ext"
        根据需要取消相应模块(.dll文件)前的注释，如

            extension=phpmysqli.dll
            extension=php_curl.dll
            extension=php_ftp.dll
            extension=php_gd2.dll
            extension=php_openssl.dll
            extension=php_pdo_mysql.dll


    5. 将 php 添加到 Apache 配置中

        打开Apache配置文件：httpd.conf(通常位于)（Apache2.2/conf/httpd.conf)
        添加下面语句行（如果不存在的话）

            LoadModule php7_module "c:/php/php7apache2_4.dll"
            PHPIniDir "c:/php"
            AddType application/x-httpd-php .php


    6. 重启apache
