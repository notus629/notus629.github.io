WebSocket

    HTML5 推出的 WebSocket，真正实现了 Web 的实时通信，使 B/S 模式具备了 C/S 模式的实时通信能力。

    WebSocket 的工作流程是这 样的：浏览器通过 JavaScript 向服务端发出建立 WebSocket 连接的请求，在 WebSocket 连接建立成功后，客户端和服务端就可以通过 TCP 连接传输数据。因为 WebSocket 连接本质上是TCP连接，不需要每次传输都带上重复的头部数据，所以它的数据传输量比轮询和 Comet 技术小了很多。
