移动端跨平台开发流派

    Web 流 （Hybrid 技术）    基于 Web 相关技术来实现界面及功能

    代码转换流                将某个语言转成 Objective-C、Java 或 C#，然后使用不同平台下的官方工具来开发

    编译流                    将某个语言编译为二进制文件，生成动态库或打包成 apk/ipa/xap 文件

    虚拟机流                  通过将某个语言的虚拟机移植到不同平台上来运行
