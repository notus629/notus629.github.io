远程访问，出现错误

    ERROR 2003 (HY000): Can't connect to MySQL server on '10.8.28.100' (10061)



问题解决

    1. 确认客户端用户访具有从localhost之外主机访问mysql的权限

    2. 查看 iptable 是否禁用端口



4. 查看端口占用情况

    $ netstat -ntlp

    或 $netstat -an | grep 3306

    确定mysql 的 3306端口已开启
    但显示的是 127.0.0.1:3306, 表示mysql的3306端口只是监听本地的连接，需进入配置文件修改绑定IP

    /etc/mysql/my.cnf

    ind-address = 127.0.0.1    注释，或者修改为远程客户端主机的IP

3. my.cnf 配置中

  注释 skip-networking
  开启 skip-networking 选项可以彻底关闭mysql的TCP/IP连接方式
