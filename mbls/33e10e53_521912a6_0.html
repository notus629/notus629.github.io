display_errors 和 error_reporting 配置

    a. 直接在 php.ini中修改

        display_errors = On 
        error_reporting = E_ALL  


    b. 可以在应用目录下建立一个 .htaccess 文件，内容如下：

        php_flag display_errors Off
        php_value error_reporting 0 （表示任何错误和提示都不会显示）


    c. 也可以直接在代码中赋值

        error_reporting(E_ALL);
        ini_set( 'display_errors', 'On' );

        error_reporting(0);
        ini_set( 'display_errors', 'Off' );
