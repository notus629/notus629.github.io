邮件发送的两种方案

    方案一 使用 php 内置的 mail() 函数，需要配置与 SMTP 服务器(无需验证的)进行通信，不支持 html 邮件和附件

    方案二 使用第三方 PHP 组件，与一般的商业 SMTP 服务器（一般须验证）进行通信来发送邮件

    方案三 直接使用 PHP 封装的 SMTP 服务器 ??



SMTP 服务器程序

    Linux: sendmail， Qmail

    Windows: hMailServer， sendmail， 自带的IIS好像也可以



PHP 封装的邮件发送组件

    PEAR： Mail_Mime, Mail

    Composer: PHPMailer(推荐，Notus), swiftmailer



ezmlm

    邮件列表管理器

    ezmlm is an easy-to-use, high-speed mailing list manager for qmail.
