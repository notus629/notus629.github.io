函数表达式

    若将函数放在本该表达式待的地方，它将被当作表达式对待，即函数表达式。
       

    在函数表达式中，名字经常被省略，没有名字的函数即匿名函数。

        // 函数表达式的使用，要在调用 area 函数之前(与普通函数的声明区分)               
        var area = function(width, height){
            return width * height;
        }
        var size = area(3, 4);



立即调用函数表达式(IIFE)

    没有名称，解释器经过时调用一次

        var area = ( function() {
           var width = 3;
           var height = 2;
           return width * heigth;
            } () ); // 或
       
        var area = ( function() {
           var width = 3;
           var height = 2;
           return width * heigth;
           } )();
