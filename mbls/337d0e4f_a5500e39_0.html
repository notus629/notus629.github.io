1.8        理解变量作用域
6种作用域规则：
*        内置超级全局变量：脚本任何地方
*        常量：全局，包括函数内部
*        脚本中声明的变量：全局，函数内部除外
*        函数内部声明的全局变量：名称与外部全局变量一致
*        内部创建并声明的静态变量：仅内部可见，多次执行中保持该值
*        函数内部创建的变量：内部有效

超级全局变量完整列表(p16)，举例
*        $GLOBALS， 所有全局变量数组
*        $_SERVER，服务器环境变量数组
*        $_GET, $_POST
*        $_COOKIE，cookie变量数组
*        $_FILES，与文件上传相关的变量数组
*        $_ENV, 环境变量数组
*        $_REQUEST，所有用户输入的变量数组

1.11        使用可变函数
 1.11.1 测试和设置变量类型
混合数据类型(mixed):被函数用作参数类型，伪"混合"，表示可以用任意数据类型。
  1. string gettype(mixed var);        //返回变量类型的字符串，或"unknown type"
  2. bool settype(mixed var, string type); //改变变量类型
其它类型测试函数
  1. is_array()
  2. is_double(), is_float(), is_real()        //检查是否浮点数
  3. is_long(), is_int(), is_integer()        //检查是否整数        
  4. is_string(), is_bool(), is_object(), is_resource(), is_null()
  5. is_scalar()        //是否标量（即整数、布尔值、字符串或浮点数）
  6. is_numberic()      //是否任何类型的数字或数字字符串
  7. is_callable()      //是否是有效的函数名称

 1.11.2 测试变量状态（p26）
bool isset(mixed var) //变量是否存在，可为逗号分隔的变量列表，全设置为true
void unset(mixed var) //销毁变量或变量列表
empty() // 变量存在且它的值非空和非0，返回false(echo 打印为空白)

 1.11.3 变量的重解释(p27)
转变变量的数据类型,把变量值转换成适当的类型返回
