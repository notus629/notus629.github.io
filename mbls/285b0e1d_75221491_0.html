Debian/Ubuntu 安装 Redis 环境 [2]

    1. 安装 redis server

        方法 a. 直接利用 Ubuntu 安装源安装

            $ apt-get install redis-server

            缺点：可能安装到旧版 Redis


        方法 b. 直接使用源码来编译并安装 Redis

            1. 安装 Redis 所需的构建工具
            2. 下载最新的 stable 版本的 Redis 源码
            3. 解压源码、编译并安装, 启动 Redis

$ sudo apt-get update                              ## 更新源列表
$ sudo apt-get install make gcc python-dev         ## 安装 Redis 所需的构建工具
 
$ tar zvxf redis-x.x.x.tar.gz
$ cd redis-x.x.x/
$ make                                             ## 注意观察编译消息，不要有错误
 
$ redis-server redis.conf                          ## 启动 Redis 服务器

## 下载并安装名为 setuptools 的简单辅助包 (simple helper package), 此包类似 pip 
 
## 下载 ez_setup 模块，通过 ez_setup 模块来下载并安装 setuptools
$ wget -c http://peak.telecommunity.com/dist/ez_setup.py
$ sudo python ez_setup.py
 
## 通过运行 setuptools 的 easy_install 包来安装 redis 包及 hiredis 包
## redis 包为 Python 提供了一个连接至 Redis 的接口
$ sudo python -m easy_install redis hiredis

Debian 类系统的快速安装 [3]

    下载压缩包，解压压缩包，执行 

        $ make && sudo make install
        $ sudo python -m easy_install redis hiredis
