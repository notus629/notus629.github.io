第4章        字符串操作与正则表达式
4.1        示例：智能表单邮件
mail()函数
bool mail(string $to, string $subject, string $message, [string  $additional_headers [, string $additional_parameters]]);
*        $message: 消息体，每行以\r\n分隔，每行最长不超过70个字符。
$message = "Line 1\r\nLine 2\r\nLine 3";
$message = wordwrap($message, 70, "\r\n");
*        在windows中php与STMP服务器交互时，行开头的句号会被移除，可以采用下面语句来解决。
$text = str_replace("\n.", "\n..", $text);
*        $additional_headers:额外的有效的文件头，见RFC2822文档, 例如"From:", "Reply-To:", "Cc:"域等，多个文件头用换行符\n\r分开
$additional_header = "From: webserver@example.com\r\n"
                                       ."Reply-to: bob@example.com";
*        When sending mail, the mail must contain a From header. This can be set with the additional_headers parameter, or a default can be set in php.ini.
使用mail()函数，必须将php设置为指向邮件发送程序。

4.2        字符串的格式化
4.2.1 字符串的整理：chop(), ltrim(), trim(), rtrim()
trim()函数
*        去除开始和结束位置的空格，并将结果字符串返回
*        空格包括：换行符\n, 回车符\r，水平和垂直制表符\t\x0B, 字符串结束符\0和空格。
*        第二个参数可提供要过滤的特殊字符
ltrim()函数，从字符串的开始处（左边）除去空格。
rtrim()函数，从字符串的结束处（右边）除去空格。
chop()函数，alias of rtrim()
4.2.2 格式化字符串以便显示（p79）
nl2br()函数，html格式化，将换行符转换为XHMTL中的<br />
printf()函数，将格式化的字符串输出到浏览器中。
sprintf()函数，返回一个格式化了的字符串。
*        转换说明（%s）的格式
%['padding_character][-][width][.precision]type
*        以%开始，打印%，必须使用%%
*        padding_char可选，默认填充空格，若非空格或0，其它填充字符前须加'为前缀
*        type,类型码，如s,u等(p80)
*        带序号的参数方式，% 后添加参数位置，并以 $ 符号结束："%2\$.2f"（即用格式说明符后第二个参数来替换）
vprintf()， vsprintf()：格式字符串和参数数组。
4.2.3        改变字符串中字母的大小写（p81）
strtoupper()：将字符串转换为大写
strtolower()：将字符串转换成小写
ucfirst()：如果字符串第一个字符是字母，就将该字符转换为大写
ucwords()：将字符串每个单词的第一个字母转换为大写
4.2.4        格式化字符串以便存储（p81）
字符插入DB中时，可能会被解释成控制符，包括引号（单引、双引），反斜杠（\），和NULL字符。
须利用 \ 进行转义，\\ 转义为\\\\
addslashes()函数，将字符串重新格式化，即特殊字符加转义。
stripslashes()函数，移除反斜杠。
magic_quotes_gpc配置指令，默认启用，使php自动添加或去除反斜杠。
*        gpc即GET, POST和cookie
*        来自gpc方法或方式的变量被自动包括在引号内。
get_magic_quotes_gpc()函数，检查系统上的这个指令是否已启用。

4.3        用字符串函数连接和分割字符串
4.3.1        使用explode(), implode()和join() （p83）
explode()函数，
array explode(string separator, string input [, int limit]);
*        根据指定分隔符将字符串分割成小块，返回到数组中
*        limit参数，限制分成字符串小块的数量。
implode()和join()函数，与explode()相反效果，两函数效果一致。
4.3.2        使用strtok()函数 (p84)
strtok()函数，一次只从字符串中取出 一些片段（即令牌）
string strtok(string input, string separator);
*        separator可为字符或字符串
*        输入的字符串会根据分隔字符串中的每个字符来进行分割。
strtok函数的调用方式
*        得到第一个令牌，调用strtok，并带两个参数
*        得到后续令牌序列，只使用separator参数,strtok会保持自己的内部指针在字符串中的位置。
*        若要重置指针，再次调用带两个参数的strtok
4.3.3        使用substr()函数 (p84)
substr()函数
string substr(string string, int start[, int length]);
*        访问一个字符串的给定起点和终点的子字符串
*        返回子字符串的副本
*        仅使用一个正数起点：返回起点到字符串结束的整个字符串
*        仅使用一个负数起点：得到字符串发问的一个子串，最后一个字符位置为-1
*        length，指定返回字符的个数

4.4        字符串的比较（p85）
4.4.1        字符串的排序：strcmp(), strcasecmp()和strnatcmp()
strcmp()函数
int strcmp(string str1, string str2);
*        相等返回0，str1按字典顺序在str2后面，返回正数，否则返回负数
*        区分大小写
strcasecmp()函数，不区分大小写，其它同strcmp().
strnatcmp()函数，按"自然顺序"比较字符串，（如2小于12）.
strnatcasecmp()函数，不区分大小写，其它同strnatcasecmp().
4.4.2        使用strlen()测试字符串的长度
e.g., strlen("Hello");

4.5        使用字符串函数匹配和替换子字符串
*        explode()和strtok()之后再通过"=="或函数strcmp()进行比较
*        还可以调用字符串函数或正则表达式来完成相同的操作
4.5.1        在字符串中查找字符串：strstr(), strchr(), strrchr()和stristr()
strstr()（等价于strchr()）
string strstr(string haystack, string needle);
*        在一个较长的字符串中查找匹配的字符串或字符。
*        找到目标关键字的一个精确匹配则返回目标关键字及其后的字符串，否则返回false
stristr()，不区分字符大小写，其它同strstr().
strrchr()，返回关键字及其最后一次出现到字符串末尾的字符串，其它同strstr().
4.5.2        查找子字符串的位置：strpos(), strrpos()(p87)
strpos()函数
int strpos(string haystack, string needle [, int offset]);
*        建议使用strpos()来替换strstr()，因为其速度更快
*        返回的整数代表搜索字符串中第一次出现目标关键字的位置，第一个位置为0
*        offset，指定被搜索字符串的开始搜索位置
*        关键字不在字符串中返回false，应使用 === false来测试返回值。
strrpos()函数，返回被搜索字符串中最后一次出现目标关键字子符串的位置，其它同strpos().
4.5.3        替换子字符串：str_replace(), substr_replace()
str_replace()，最常用字符串替换函数
mixed str_replace(mixed needle, mixed new_needle, mixed haystack [, int & count]));
*        用new_needle替换所有haystack中的needle
*        返回替换后的结果
*        count包含了要执行的替换操作的次数
*        可传递数组，被替换单词、替换单词、返回均为数组
substr_replace()，在给定位置中查找和替换字符串中选定的子字符串。
string substr_replace(string string, string replacement, int start int [length]);
*        length值为0，则替换字符串实际上会插入到字符串中而不覆盖原有的字符串
*        正的length表示用新的字符串替换掉的原字符串的长度
*        负的length表示从字符串尾部开始到第length个字符停止替换
*     不给定length值表示从字符串start位置开始一直到字符串结束。

4.6        正则表达式的介绍
php支持两种正则表达式语法：POSIX和Perl（PCRE, Perl Compatible Regualr Expression）
php5.3版本中Perl风格不能被禁用，此处介绍POSIX语法
POSIX更简单，但是它们不是二进制安全的
4.6.1        基础知识
正则表达式：一种描述一段文本模式的方法
精确匹配
用特殊字符指明表达式的元意（meta-meaning）:开始结尾模式、模式的部分重复、模式中的字符属于特定的某一类型、按特定字符的出现来匹配。
4.6.2        字符集和类（p89）
字符集：用于匹配属于特定类型的任何字符
.字符：作为一个通配符来代替除换行符(\n)外的任何一个字符。
方括号（[]）中的内容：是一个字符类，一个被匹配字符所属的字符集合。其中的表达式只匹配一个字符。
*        如：[a-z]at, [aeiou]
*        描述一个范围:[a-zA-Z]
*        指明字符不属于某个集[^a-z]
*        预定义的字符类：用于POSIX风格的正则表达式的字符类（P90）
4.6.3        重复
符号"*"，表示这个模式可以被重复0次或更多次，放在要作用的表达式的后面。
符号"+"，表示这个模式可以被重复1次或更多次，放在要作用的表达式的后面。
4.6.4        子表达式
可将一个表达式分隔为几个子表达式，
*        例如：表示"至少这些字符中的一个需要精确匹配"
使用圆括号来实现子表达式，如
(very)*large
4.6.5        子表达式计数（p91）
花括号{}中的数字表达式来指定内容允许重复的次数
确切的重复次数，如{3},重复3次
重复的次数范围，如{2,4},重复2~4次
开底域的重复范围（{2,}表示至少重复两次）。
4.6.6        定位到字符串的开始或末尾
可以确定一个特定的子表达式是否出现在开始、末尾或在两个位置都出现。
脱字符号（^）：用于正则表达式的开始，表示子字符串必须出现在被搜索字符串的开始处。
字符（$）：用于正则表达式的末尾，表示子字符串必须出现在字符串的末尾。

特殊字符：., $，\等，前面加上\
php中必须将正则表达式模式包括在一个单引号字符串中。
双引号引用的正则表达式带来复杂性：

4.7        用正则表达式查找子字符串(p93)
POSIX风格：ereg()和eregi()
int ereg(string pattern, string search, array [matches]);
*        搜索字符串search，如果找到了与pattern匹配的字符串，则存储在matches中
eregi()，不区分大小写，其余同ereg
4.8        用正则表达式替换子字符串(p94)
ereg_replace(), eregi_replace()
string ereg_replace(string pattern, string replacement, string search)
在search中查找正则表达式pattern的字符串，用replacement来替换
eregi_replace()，不区分大小写，其余同ereg_replace()
4.9        使用正则表达式分割字符串(p94)
array split(string pattern, string search [, int max]);
*        将search分割为符合正则表达式模式pattern的子串，将子字符串返回到数组中
*        max指定进入数组中的元素个数
同样的功能，正则表达式函数运行效率低于字符串函数
