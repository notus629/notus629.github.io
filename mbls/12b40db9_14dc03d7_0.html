git rebase -i

    压缩历史

    ## 一次完成 git add 和 git commit 操作
    $ git commit -am "Add feature-C"

    ## 选定当前分支中包含 HEAD （最新提交）在内的两个最新历史为对象，并在编辑器中打开
    $ git rebase -i HEAD~2


示例

    ## 合并上两次提交
    $ git rebase -i HEAD^^
    $ git rebase -i HEAD~2

    若出现 interactive rebase already started, 表示在中途出了问题，只需

    $ git rebase -i --abort
