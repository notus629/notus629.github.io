基础JavaScript脚本
       扩展名 .js
       CSS文件通常应放置到一个名为CSS或style的文件夹中
       JavaScript文件通常被放置到名为scripts, javascript或js的文件夹中


从HTML页面链接JavaScript脚本
       <script src="path/*.js"> </script>

将内容写到HTML页面中
       document.write("...");        
           // 将新内容写入页面中放置<script></script>的地方
           // 非最佳方法
       对象的成员：方法、属性
       点号：成员操作符

JavaScript不会修改HTML标记，脚本是和创建的Web页面的模型进行交互的。
