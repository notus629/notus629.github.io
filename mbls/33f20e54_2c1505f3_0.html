error_reporting

    display_errors string

    This determines whether errors should be printed to the screen as part of the output or if they should be hidden from the user.

    "stderr" sends the errors to stderr instead of stdout.



set at runtime

    ini_set( 'display_errors', 'Off' );

    default value: 1



Note:

    Although display_errors may be set at runtime , it won't have any effect if the script has fatal errors. This is because the desired runtime action does not get executed.
