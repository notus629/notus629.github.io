第2章 Linux 安装


        引导加载器：Grub(多重引导程序，Linux默认)，早期使用LILO引导工具，可以引导Linux, Windows.

        虚拟机：宿主操作系统(Host OS) -> 虚拟机软件 -> 客户操作系统(Guest OS)

        免费虚拟机软件：VMWare Player（VMware Workstation Pro的简化版，免费）,Virtual Box(开源)


硬盘分区

        硬盘分类：IDE, SCSI, SATA

        IDE 接口设备被称为 hd，SCSI和SATA接口设备被称为sd，第一块硬盘被称为sda,第二块sdb;

        Linux 规定一个硬盘上只能存在4个主分区，分别(SCSI和SATA类)命名为sda1, sda2, sda3,sda4.

        逻辑分区从5开始标识，没有数量限制。

        每个系统都需要一个主分区来引导，操作系统主体可以安装在主分区也可以安装在逻辑分区，但引导程序必须装在主分区。

        安装要点：新建分区表后再添加，分区1（主分区）挂载点/，Ext4日志文件系统。交换空间swap，无挂载点，该分区相当于虚拟内存，用于缓冲数据。

        Ubuntu Linux中，现在设置的用户具有管理员权限，而Red Hat、SUSE需要另外设置一个root.



进阶：修复受损的Grub


        安装双系统时，一定要先安装Windows再安装Linux，因为Windows安装时会将多重引导程序Grub覆盖。而Windows的引导程序并不能引导Linux.

        修复方式：利用安装光盘启动计算机，进入Rescue System(救援系统模式)，重新安装Grub(P24-25)

        Grub不区分IDE,SCSI或SATA，统一表示为hd0, hd1等。（hdn,m），表示第n+1块硬盘上的第m+1个分区
