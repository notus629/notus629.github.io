pear的安装使用-Windows [2]

    方式一： 若 php 目录下有 c:\php\go-pear.bat，执行即可

    方式二： 下载 go-pear.phar, 执行 $ php go-pear.phar 开始安装


    然后将安装目录添加到 PATH 环境变量，或者安装好后在 PHP 目录下找到  PEAR_ENV.reg，双击即可



Linux

    通常 PHP 时已经安装，除非在编译时配置 ./configure 选项 --withoutu-pear


    安装命令

        $ wget http://pear.php.net/go-pear.phar
        $ php go-pear.phar



基本使用

    ## 安装 Mail_Mime 包
    $ pear install Mail_Mime

    ## 查看当前系统已经安装的 PEAR 包
    $ pear list

    ## 查看已安装的 Mail_Mime 包安装的文件
    $ pear list Mail_Mime

    ## 查看已安装的 Mail_Mime 包的最新版本
    $ pear upgrade Mail_Mime

    ## 查看所有可供安装的包名称
    $ pear list-all
