hmailserver [1]

    开源，仅 Windows



安装

    要设置密码，用于管理 hmailserver

    需要使用数据库系统，选择 MySQL 相关

    需要创建 hMailServer 数据库， database server type: MySQL


    设置 hMailServer 服务依赖，选择依赖的数据库服务

        为了确保数据库服务的正确

        此处 ( wampserver3.0.6 64bit , win10) 选择： wampmysqld64 (wampmysqld64)
        

    提示，The MySQL client needs to be manually copied to the hMailServer Bin directory

        即将 libmysql.dll 拷贝到： C:\Program Files (x86)\hMailServer\Bin\libmysql.dll

        注意：wamp64对应的为64位的，此处需要的是 32bit 的，得装一个 wamp32bit,然后找到其中的 libmysql.dll


设置

    在本地使用：直接在 welcome 项中，add domain，填 localhost，勾上 enabled 即可

        由于此处在 php.ini 中填写的，sendmail_from=bugber@localhost, 若此处填的不是 localhost，会出现 “SMTP server response: 550 Unknown user ” 错误。

        注1：只需要保持 hmailserver 中的域名与 php.ini 中设置的 sendmail_from 中的 email 的域名一致即可（可与 smtp 服务器地址不一致），例：
            ;php.ini 配置
            SMTP = localhost
            smtp_port = 25
            sendmail_from ="bugber@notus.com"
            ;hmailserver 设置
            hmailserver domain: notus.com
        
        注2：若发件人与收件人的域名一致，也会出现 550 错误。

        注3：似乎不太稳定，发送给 bugber@bugber.com 能成功，但发送给 hehe_xiao@qq.com 的始终不成功

    其它不用填。

    最好把防火墙（系统，360等）关掉，测试成功
