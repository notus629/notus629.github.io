常用命令

命令基本功能
ar(archive) ar rcs 将多个二进制目标打包成静态库
bash -c
/bin/sh -c		bash –c command 或 /bin/sh –c command 启动一个shell
并运行command命令
cp将一个档案拷贝至另一档案，或将数个档案拷贝至一个目录
cp [options] source dest
cp [options] source ... directory
curl下载命令， ubuntu 16.04自带，其余版本似乎也有
df -h查看磁盘空间，-h (human readable)
find搜索
$ find / -name "ifconfig"
exit退出当前登录用户
group添加用户组
history查看shell中执行命令的历史记录
id查看用户的UID、GID及其所属的组
显示当前登录用户的信息（不带参数）
ifconfig查看网卡信息
kill向进程发送一个信号
ldd查看一个二进制可执行程序或共享库所依赖的共享库
ln创建硬连接或符号连接（文件目录均可）
ln [option] ... 源文件  目标文件(即连接文件）
-s, --symbolic    创建符号连接而非硬连接
lsof查看占用文件的进程（指定的文件或所有打开文件）
nm列出库中的符号清单
passwd修改密码
（设置/修改root密码：$sudo passwd root)
ps查看进程的树状层次关系
reboot重启
scp在2个linux主机间复制文件
格式：scp[可选参数] file_source filetarget
startx在字符界面下会转到图形环境
（Ubuntu下会变成一个不完整的图形界面，问题？？？）
size查看一个可执行程序的正文段、数据段、bss段的长度（字节数
su用户切换（带用户名台数）
切换到root(不带参数）
Ubuntu也可以使用，若不记得su后的root密码，可以用sudo su,只需输入当前用户密码
sudo放在命令之前，以root的身份去执行
top即时跟踪当前系统中进程的情况
uname不带选项：(unix name), Linux系统下显示为: Linux
-r, --kernel-release 内核版本号，如 2.6.18-398.e15
-a 更全的信息
useradd添加用户
userdel删除用户账号
usermod修改用户帐号
visudo修改sudoers文件（会进行额外的处理）
