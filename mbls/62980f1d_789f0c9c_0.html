获取元素内容

    html() 和 text() 方法

        可以获取和更新元素的内容



html()

    返回第一个匹配元素内部的 HTML，包括其所有的后代节点（不包括匹配元素本身）



text()

    返回选取结果中每个元素的内容（不含标签，不仅仅是第一个），包括所有后代元素中的文本。



注：

    1. input 和 textarea 元素的内容，使用 .val() 方法

<script type="text/javascript">
    $(function(){
         var $list = $('ul');
         var listText = $list.text();
         $list.append('<p>' + listText + '</p>');
    });
</script>
