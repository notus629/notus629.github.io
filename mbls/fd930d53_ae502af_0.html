开放虚拟文件格式（OVF）

    Virtual Box 菜单：管理 -> 导入虚拟电脑，即可导入 OVF 格式文件

    在导入对话框的文件后缀中，支持:开放式虚拟化格式（*.ova, *.ovf)



虚拟硬盘

    Virtual Box 菜单：新建虚拟电脑 -> 填好“虚拟电脑名称和系统类型” -> 设置分配的内存大小 -> 在虚拟硬盘界面，选择“使用已有的虚拟硬盘文件” –> 载入 Ubuntu.vdi 文件。

    导入时不能有同样的(即使是复制的）vdi创建的虚拟机，否则会报错。

    导入对话框的文件后缀中，支持：全部虚拟硬盘文件（*.vmdk, *.vdi, *.vdh, *.hdd, *.qed, *.qcow, *.qcow2, *.vhdx）



虚拟硬盘文件类型的选择

    创建虚拟机时，可以进行选择，默认为 VDI

        VDI （VirtualBox 磁盘映像）
        VMDK （虚拟机磁盘）
        VHD （虚拟硬盘）
        HDD （并口硬盘）
        QED (QEMU 增强型磁盘）
        QCOW (QEMU 写入时复制）
