代码折叠

    Ctrl + Numpad + / -              打开/折叠选中的代码段

    Ctrl + Shift + NumPad+           折叠所有

    Ctrl + Shift + NumPad-           开启所有折叠



搜索

    Ctrl + N                         寻找一个类

    Ctrl + Shift + N                 寻找一个文件

    Ctrl + Alt + Shift + N           寻找一个符号



书签

    Alt + Home                       显示文件层级浏览工具

    F11                              开启/关闭书签

    Ctrl + F11                       开启/关闭数字书签

    Ctrl + #[0-9]                    中转到数字书签

    Shift + F11                      显示所有的书签



Refactor/Rename

    Shift + F6                       rename classes, methods and variables with automatic correction of all places where they are used.



注释 

    ctrl + shift + /                   块注释

    /** 回车                           块注释

    ctrl + /                           行注释



Ctrl + Alt + j

    用 html 标签来包围光标处的内容



Ctrl + Alt + L

    格式化内容



Ctrl + o

    找类中的方法（必须是一个 php 类）
    即，方法大纲



按 Ctrl 键不放，再单击

    代码追踪



加上 // TODO 注释

    在代码完成后会提示未完成的 TODO 内容





ctrl + D 复制当前行到下一行


alt + F1 当前选中目标在几种视图之间切换

ctrl + W 选取光标所在的词、区域、更大区域...
