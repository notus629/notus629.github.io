Windows 上安装 PHP 扩展

    两种方式：编译到 PHP 中，或者动态载入 DLL，后一种为推荐的方式。


    PHP 扩展 （php extension），命名方式 "php_*.dll"，* 表示扩展名

        如 mysqli 扩展: php_mysqli.dll



分两个步骤

    a. 下载对应版本的 dll 库文件，放置到对应的 PHP 库文件存放目录 "php\ext" 下

        须注意：dll should be complied with the same compiler the php is.
        （如，同为 vc9, vc8 等，在 phpinfo compiler项中可以查看编译器版本）


    b. 在 php.ini 配置文件中，设置载入的库文件，以及库文件的存放目录，示例如下

        extension = php_extname.dll   ; dll 扩展名
        extension_dir = ...           ; setting the location of DLLs 

        也可以在 PHP 脚本中利用 dl() 函数来动态载入



检查是否正确安装

    在 phpinfo 中查看，能找到,如 mysqli 条目，表明已正确安装 php_mysqli.dll 扩展
