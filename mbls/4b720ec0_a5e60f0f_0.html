不同的事件类型

    W3C DOM 事件 （W3C DOM 规范）

        由 W3C 管理制定的，本章的大多数事件都属于此规范，如

        按下了哪个键，鼠标在什么位置


    HTML5 事件 （HTML5 规范）

        HTML5 规定的浏览器应该支持的和 HTML 相关的事件，如

        表单提交或表单元素变化时应该触发的事件

            submit

            input

            change

            readystatechange

            DOMContentLoaded

            hashchange


    BOM 事件

        作为浏览器对象模型(BOM)的一部分，浏览器厂商实现的一些相关的事件, 如用于触屏设备的

            touchstart

            touchend

            touchmove

            orientationchange
