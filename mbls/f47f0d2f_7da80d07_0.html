程序示例 (first.sh)

    #! /bin/bash
    cd /tmp
    echo "Hello, world"



三种运行方式

    赋予程序可执行权限直接运行

        
        $ chmod a+x first.sh
        $ ./first.sh


    调用命令解释器(shell)执行

        $ bash first.sh


    使用source命令(点命令，也可使用.)执行

        source为内部命令，不会创建新shell，会影响当前shell

        $ source first.sh
        $ . first.sh



Shell 执行命令的顺序（由高到低）

    别名            由"alias command=..."创建
    关键字          如: if, for, while等
    函数            shell 语言书写的代码
    内部命令        shell 本身的命令 cd, echo, source等
    外部命令        二进制可执行程序或脚本程序



内部命令 type

    查看所要执行的是上述哪种命令

    $ type ls
