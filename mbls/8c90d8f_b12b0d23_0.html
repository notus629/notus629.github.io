如何将 session 数据放入到 memcached (针对 php memcache 扩展)

    默认是将 session 放入到文件


    修改 php.ini

        session.save_handler = files | user | memcache
            files:  默认值，放入到某个路径（由 session.save_path 设置 )
            user: 自定义
            memcache


    设置如下

        session.save_handler = memcache
        session.save_path = "tcp://127.0.0.1:11211"
