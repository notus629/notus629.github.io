character set 

    字符集。



collation

    用于指定数据集如何排序，以及字符串的比对规则

    某种特定的 character set 可能会对应多个不同的 collation



查看 collation

    mysql > show collation;



collation 名字的规则

    可以归纳为这两类：
        1. <character set>_<language/other>_<ci/cs>
        2. <character set>_bin

    例如:
        utf8_danish_ci
        utf8_general_ci (可能表示通用，不分语言）
 
    ci 是 case insensitive 的缩写
    cs 是 case sensitive 的缩写。即，指定大小写是否敏感。
    (奇怪的是 utf8 字符集对应的 collation 居然没有一个是 cs 的)
