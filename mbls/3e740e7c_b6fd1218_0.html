CentOS 7 环境配置



EPEL
    
    EPEL( Extra Packages for Enterprise Linux ) 是 yum 的一个软件源，里面包含了许多基本源里没有的软件

    EPEL 会有很多源地址，如果一个下不到，会去另外一个下

    适合 Fedora, RHEL, CentOS



安装步骤


    1. 配置 YUM 源，安装 EPEL.

        CentOS 7 的 默认 YUM 源里的软件包版本可能不是最新的，如果要安装最新的软件包就得配置下 YUM 源。

        $ yum install -y epel-release


    2. 配置 MySQL YUM 源

        ## 安装 RPM 包前需要导入 RPM-GPG-KEY 文件 (实际 rpm 命令不成功，改用 GPG 命令)
        ##  mysql_pubkey.asc 文件参考[]
        $ gpg --import mysql_pubkey.asc
        $ rpm --import mysql_pubkey.asc

        ## 导入后安装 CentOS 7 的 MySQL RPM 包 (或直接下载*.rpm文件来安装，参见[3])
        $ rpm -Uvh http://repo.mysql.com/mysql57-community-release-el7-9.noarch.rpm
