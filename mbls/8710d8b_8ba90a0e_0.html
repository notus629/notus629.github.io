git checkout, reset and revert [1]

    撤销 repo 的一些操作

    reset 和 checkout 既可以用于 commit 级别，也可以用于 file 级别。

    git reset 和 git checkout的参数会决定命令的作用范围。当命令并不包括含一个文件路径时，命令作用于整个 commit。



commands scope common user cases
git reset    Commit         Discard commits in a private branch or throw away uncommited changes
git reset    File     Unstage a file
git checkout    Commit     switch between branches or inspect old snapshot
git checkout     File     Discard changes in the working directory
git revert    commit     Undo commits in a public branch
git revert    File     不支持
