实现下拉菜单

    需要安装 superfish [1] 这个第三方模块 



安装启用 superfish 模块

    管理 -> 扩展 -> 安装新模块


    下载 superfish 支持库文件 [1]，将下载的 Superfish-for-Drupal-2.x 压缩文件解压到网站的指定目录(改名为Superfish)

        \sites\all\libraries\Superfish (无此目录则创建）


    查看是否正确安装了 superfish 模块及相应的支持库文件

        管理 -> 报告 -> 报告状态页面，可查看到 Superfish library 已安装信息



配置菜单选项，实现下拉菜单显示

    结构 -> 区块布局 -> 主导航 -> 配置 -> 设置菜单层级 (安装 Superfish 插件后才出现)
