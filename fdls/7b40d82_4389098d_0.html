设置首页

    禁用 logo, 点击页面右上角“编辑”，然后进行后续操作

    搜索和工具区块类似操作。

    
    让首页不显示“主内容”区域。

        \themes\business\templates\。编辑该文件，找到第124行左右，将{{ page.content }}前后加上判断语句，变成

        {% if not is_front %}
            {{ page.content }}
        {% endif %}


    在首页轮播图片下添加“公司简介”区块

        通过管理菜单导航到“结构>区块布局”页面。点击“自定义区块”页签

        再点击“添加自定义区块”按钮。输入“区块描述”和“正文”的内容

        通过管理菜单导航到“结构>区块布局”页面。在“区块布局”页签所在的页面，找到Home Quotes区域，点击其右边的“放置区块”按钮，将创建的区块放进去。


    在首页“公司简介”区块下面添加三个图文区块

        先添加自定义区块

        在“区块布局”页签所在的页面，找到Home Hightlight 1、Home Hightlight 2、Home Hightlight 3区域，点击其右边的“放置区块”


    在首页添加四个文字区块

        添加自定义区块

        在“区块布局”页签所在的页面，找到FIRST BOTTOM、SECOND BOTTOM、THIRD BUTTOM 、FOURTH BUTTOM区域，点击其右边的“放置区块”


    在首页四个文字区块下面再放置一个单列文字区块

        在“页脚”区域，放置自定义的区块（取消勾选“显示标题”）


    设置“企业简介”区块只在首页显示

        编辑模式->点击该模块右上角的“配置区块”->页面标签，在页面textarea中输入<front>
