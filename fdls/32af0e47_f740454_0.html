位置

    laravel/resources/views/



视图文件

    普通的模板，命名为 xxx.php

    Laravel 默认 blade 模板，命名为： yyy.blade.php

    laravel 中的模板支持原生的 php 文件



控制器中输出模板

    方法中, return view('xxx');

    输出默认 blade 模板，则 return view('yyy');



一般一个控制器对应 views 下一个目录

    方法中输出模板，则 return view('member/xxx');

    或 return view('member.xxx');



模板带变量

    方法中

        return view('member/xxx', [
            'name' => '天秤vs永恒', 
            'age' => 18
        ]);


    模板中输出变量

        {{$name}}, {{$age}}
