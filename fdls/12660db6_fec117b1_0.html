操作每个选取结果 - each()方法

    .each() 方法

        在选取结果上实现循环功能

        参数： 函数，需要在每个元素上运行的代码，命名函数或匿名函数


        // this 指向当前节点，$(this) 包含当前元素的 jQuery 对象
        $('li').each(function(){
            var ids = this.id;
            $(this).append('<em class="order">' + ids + '</em>');
        });


        函数的参数 index: 每个被选中元素的索引

        $('li').each(function(index){
            $(this).hide().delay(700*(index+1)).fadeIn();
        });
