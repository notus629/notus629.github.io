v-bind 属性

    data:{
        url: '...',
    }


    写法一

        ## 效果能出来，但控制台 vue 会提示 404 请求的错误    
        <img src="{{url}}" alt="">       


    写法二， vue 方式

        ## 不会出现 404 错误
        <img v-bind:src="url" alt=""> 



v-bind 简写

    v-bind:src= 简写为 :src=, 推荐

    类似还有 :width, :title, :height 等类似
