安装

    Laravel 框架各版本对 PHP 的要求

        5.1 5.2    PHP 5.5.9+
    
        4.2        PHP 5.4+



Via Laravel Installer

    ## 1. download the Laravel installer using Composer
    $ composer global require "laravel/installer"    

    ## 2.  create a fresh Laravel installation in the directory you specify, e.g. blog
    ##   the directory blog contains a fresh Laravel installation with all of Laravel's 
    ##   dependencies already installed:
    $ laravel new blog


    注 1：跟网速有关系，国内无法下载（须使用 composer 中国镜像）

    注 2: Make sure to place the $HOME/.composer/vendor/bin directory (or the equivalent directory for your OS) in your $PATH so the laravel executable can be located by your system. (Windows 10 下为 C:\Users\leo\AppData\Roaming\Composer\vendor\bin 目录)

        

Via Composer Create-Project

    注：跟网速有关系，国内无法下载（须使用 composer 中国镜像）

    $ composer create-project --prefer-dist laravel/laravel blog



直接下载一键安装包

    下载后，重命名，拷贝到根目录即可 [2]
