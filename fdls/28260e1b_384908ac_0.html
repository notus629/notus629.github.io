版本区分

    uncompressed: is best used during development or debugging

    compressed: saves bandwidth and improves performance in production

    sourcemap file:  use when debugging with a compressed file

    slim build version:  it excludes ajax, effects, and currently deprecated code.



版本号

    1.x 与 2.x 区别   

        2.x 不再兼容 IE6、7、8浏览器，这样做的目的是为了兼容移动端开发。由于减少了一些代码，使得该版本比 jQuery 1.x 更小、更快。



各版本对浏览器的兼容性

安装

    jQuery是一个JavaScript脚本库，不需要特别的安装，只需要我们在页面 <head> 标签内中，通过 script 标签引入 jQuery 库即可。

    <script type="text/javascript" src="http://libs.baidu.com/jquery/1.9.1/jquery.js"> </script>
