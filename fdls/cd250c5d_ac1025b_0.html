PHPCMS

    版本 PHPCMS 2008, PHPCMSV9



要求的 Windows PHP 安装方式

    PHP 以 ISAPI （非 CGI）方式安装

    建议不要启用 PHP 安全模式运行



系统要求

    可用的 httpd 服务器（如 Apache, Zeus, IIS 等）

    PHP5.2或更高版本

    MySQL版本推荐5.0或更高版本
