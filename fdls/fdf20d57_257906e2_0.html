双标签

    


基本语法

    {dede:标签名 参数名 = '值' 数据名 = '值'｝
     // 内容
    ｛/dede:标签名｝



实例

    使用 channel 标签取出所有的栏目信息

    {dede: channel type='top' row='10'}
    [field:typename/]
    {/dede: channel}



标签调用究竟发生了什么？

    1. 当我们调用某个标签时，实际上引入对应的标签库，如 dede:channel, 其实就引入了 channel.lib.php(一个标签对应一个标签库）


    2. 比如从上面的这个{dede:channel}标签，相当于执行
        select * from dede_arctype where ???

        可以查看后台发生的 sql, 见 channel.lib.php


    3. 每一个标签对应一个标签库，如果我们在开发中需要自己的定制的东西，可以根据需要修改源代码
