表单元素选取

   使用这些选择器，jQuery 会检查文档中的每一个元素来进行查找，速度会比使用 CSS 选择慢。

   因先缩小查找范围，再使用此类选择器。


    :button            button 元素和 type 属性为 button 的 input 元素

    ：checked          选中的单选按钮和复选框元素

    :disabled          所有被禁用的元素

    :enabled           所有被启用的元素

    :focus             当前焦点所在的元素（性能不如 $(document.activeELement))

    :file              所有文件选择器元素

    :image             所有图片按钮

    :input             所有 button, input, select 和 textarea 元素

    :password          所有密码输入框

    :radio             所有单选按钮

    :reset             所有重置按钮

    :selected          所有被选中的元素

    :submit            button 元素和 type 属性为 submit 的 input 元素

    :text              选取所有 type 属性为 text 或未指定 type 属性的 input 元素


    .val()             主要用于 input, select 和 textarea 元素，可用来获取匹配元素中第一个元素的值，或是设置所有元素的值

    .filter()          使用第二个选择器（尤其是表单选择器）来筛选 jQuery 选取结果

    .is()              通常和筛选器一起使用，用来检查表单元素是否被选中


    $.isNumeric()      全局方法。检查值是否是一个数字，并返回一个布尔结果（如, $.isNumeric(4.4), $.isNumberic("2") 等）



事件

    .on()              用来处理所有事件

    
    blur               当元素失去焦点时

    change             当输入框中的值发生变化时

    focus              当元素获得焦点时

    select             当 select 元素的选项发生变化时

    submit             当表单被提交时
