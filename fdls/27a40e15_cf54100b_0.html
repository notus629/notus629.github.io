获取和设置CSS属性

    .css() 方法，获取或设置 CSS 属性的值



获取 CSS 属性

    // 获取 CSS 属性, 返回属性值（以 RGB 值的形式返回）
    var backgroundColor = $('li').css('background-color');



设置 CSS 属性

    // 会设置所有列表的背景色
    $('li').css('background-color', '#272727');

    // 以像素为单位的尺寸时，可使用 += 和 -= 操作符来增加或减少属性的值
    $('li').css('padding-left', '+=20');



设置多个属性

    使用对象描述标记更新多个属性，属性名加引号，每个属性和值对之间用逗号分隔

    $('li').css({
        'background-color': '#272727', 
        'font-family': 'Courier'
    });
