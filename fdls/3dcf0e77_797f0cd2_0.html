Laravel 框架各版本对 PHP 的要求

    5.1 5.2    PHP 5.5.9+
    4.2        PHP 5.4+



环境推荐

    MAC: MAMP(可切换不同的 PHP 版本)

    Windows 推荐: XAMPP

    Linux: 自行编译安装



下载 & 安装 [1]

    因被墙的原因，推荐一键安装包

    暂时不用 composer，还是墙的原因

    下载 Laravel 5.2.15, 解压并重命名为 laravel 后，拷贝到根目录即可

    访问：localhost/laravel/public
