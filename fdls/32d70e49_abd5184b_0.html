两种更新数据方式

    更新为指定内容

    自増和自减



更新为指定内容

    // 更新一定要带条件，返回影响的行数
    $num = DB::table('student')
        ->where('id', 12)
        ->update(['age' => 30]);



自増、自减（可以不带条件）

    // 全部自增, 増 1，返回受影响的行数
    $num = DB::table('student')->increment('age');    

    // 全部自増，増 3
    $num = DB::table('student')->increment('age', 3); 

    // 全部自减, 减 1, 减 3
    $num = DB::table('student')->decrement('age');
    $num = DB::table('student')->decrement('age', 3); 

    // 带条件自减
    $num = DB::table('student')
        ->where('id', 12)
        ->decrement('age', 3);

    // 自増的同时，修改其它字段
    $num = DB::table('student')
        ->where('id', 12)
        ->increment('age', 3, ['name' => 'iimooc']);
