添加联系表单

    Dropal 8 核心自带了创建联系表单功能，并且还自带了已经建立好的两个表单，个人联络表单和网站反馈表单。

    可以设置个人和网站全局表单的字段和显示方式。

    管理 -> 结构 -> 联系表单



个人联络表单

    是给网站访问者联系注册用户的表单；该表单的名称和接收人不能编辑。



网站全局联系表单

    用于访问者给您的网站邮箱发送邮件。您可以编辑名称和接收者。

    网站反馈表单中的所有字段都可以编辑。也可以添加自定义的联系表单。



将联系表单添加到主菜单链接

    本例中，“网站反馈”的机器名是“feedback”，对应的的网站内部访问路径就是 “/contact/feedback”

    管理 -> 结构 -> 菜单 -> 编辑菜单 -> 添加菜单

        菜单链接标题

        链接：/contact/feedback
