键盘事件

    @keydown="show" //()可以省略

    @keydown="show($event)"   ev.keyCode

    @keyup="show($event)"  ev.keyCode
        if(ev.keyCode == 13){
            alert('您按回车了!');
        }

    常用的键：

        回车：
            @keyup.13="show()"     // 按回车时执行
            @keyup.enter="show()"  // 同上
        上、下、左、右
            @keyup.up, @keyup.down, @keyup.left, @keyup.right
            @keydown.up, @keydown.down, @keydown.left, @keydown.right
            其它，.delete 等（abc 等没封装）
