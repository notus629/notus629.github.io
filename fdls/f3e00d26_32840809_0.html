CMS

    cms (content manage system) 内容管理系统。

    比如，电商网站/小说网/博客/文章管理系统/软件下载网/公司宣传性网站/新闻发布网站



主流的 CMS 有哪些？

    dedeCMS(织梦CMS), 帝国 CMS, PHPCMS, CMSTOP 等

    dedeCMS: 专注于个人网站/中小型网站/也有企业用户

    PHPCMS: 针对中型网站

    CMSTOP: 针对流量大，并发大的网站

    帝国CMS: 针对中小型网站



dedecms 的特点

    1. 开源、简单、实用

    2. 技术架构 PHP + MySQL

    3. 专注于个人网站和中小型的门户网

    4. 可以实现全站静态化
