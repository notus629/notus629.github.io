微擎Tpl控件函数

    系统Tpl控件函数全部位于 framework/function/tpl.func.php 文件内

    该文件内所有函数使用前必须加载文件: load()->func('tpl');



日期范围

    {php echo tpl_form_field_daterange('birth');}

     第二个参数为其缺省的值，格式：array('start'=>123, 'end'=>284)



图片上传控件

    分app端和web端，web端如下

    {php echo tpl_form_field_image('thumb');}

    第二个参数为其缺省的值



富文本编辑器

    使用 jquery，将 input 表单变成富文本编辑器

    <input  type="text" id="info" />

    <script>
        require(['jquery','util'], function($, util){
            $(function(){
                var editor = util.editor($('#info')[0]);
            });
        });
    </script>
