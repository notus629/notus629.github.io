键盘事件

    ## ()可以省略
    @keydown="show" 


    ## keydown 事件, 按键对应的键值：ev.keyCode
    @keydown="show($event)"   


    ## keyup 事件, 按键对应的键值：ev.keyCode
    @keyup="show($event)" 

    if(ev.keyCode == 13){
        alert('您按回车了!');
    }



常用的键

    回车键

        @keyup.13="show()"     // 按回车时执行

        @keyup.enter="show()"  // 同上


    上、下、左、右键

        @keyup.up, @keyup.down, @keyup.left, @keyup.right

        @keydown.up, @keydown.down, @keydown.left, @keydown.right


    其它类似的还有

        .delete 等

        abc 等键没封装
