插入元素

    <li> item </li>


    .before()         插入到 <li> 之前

    .after()          插入到 </li> 之后

    .prepend()        插入到 <li> 和 item 之间

    .append()         插入到 </li> 和 item 之间


    a.prepend(b)      b 插入到 a 中
    a.append(b)       b 插入到 a 中

    a.prependTo(b)    a 插入到 b 中
    a.appendTo(b)     a 插入到 b 中



插入的参数

    可以是 jQuery 对象，也可以是 html 元素或 DOM 元素


    jQuery 对象举例

        // 包含一个空的<li>元素
        $('<li>');    

        // 包含一个带有 class 属性和一些文本的 <li> 元素
        $('<li class="new">item</li>);
