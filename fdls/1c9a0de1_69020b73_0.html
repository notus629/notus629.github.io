01. Notus 补充笔记



视频教程环境

    wampserver 版本：wampserver2.2e-php5.3.13-httpd2.2.2-



配置 - 基本步骤

    1. 在项目目录下放置 ThinkPHP 核心目录内容（以 ThinkPHP 为根的目录树，包括 ThinkPHP 目录本身）


    2. 在项目目录（即文档根目录下的 thinkPHP 目录）下，其主入口文件为 index.php（需自建），在浏览器中运行后自动帮我们创建相应的应用文件夹

        <?php
            // 1. 确定应用名称（前台应用）
            define('APP_NAME', 'Home');
            // 2. 确定应用路径
            define('APP_PATH', './Home/');
            // 3. 引入核心文件
            require './ThinkPHP/ThinkPHP.php';


    3. 再新建一个后台应用，新建一个 admin.php 如下。运行后自动创建后台应用目录结构。
     
        <?php
            // 1. 确定应用名称（后台应用）
            define('APP_NAME', 'Admin');
            // 2. 确定应用路径
            define('APP_PATH', './Admin/');
            // 3. 引入核心文件
            require './ThinkPHP/ThinkPHP.php';


    4. 前台和后台的公有目录，可以如下方式新建

        /项目目录/Public/                // 如javascript, CSS等公有代码

        /项目目录/Uploads/               // 如图片等
