项目目录结构及说明

       Home 前台应用文件夹
       ├─Common 项目公共文件目录
       ├─Conf 项目配置目录
       ├─Lang 项目语言目录
       ├─Lib 项目类库目录
       │  ├─Action Action类库目录
       │  ├─Behavior 行为类库目录
       │  ├─Model 模型类库目录
       │  └─Widget Widget类库目录
       ├─Runtime 项目运行时目录
       │  ├─Cache 模板缓存目录
       │  ├─Data 数据缓存目录
       │  ├─Logs 日志文件目录
       │  └─Temp 临时缓存目录
       └─Tpl 项目模板目录
