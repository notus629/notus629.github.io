html5 dataset

    允许你在普通的元素标签里嵌入类似data-*的属性，来实现一些简单数据的存取。    

    数量不受限制，并且也能由JavaScript动态修改，也支持CSS选择器进行样式设置。

    能够更加有序直观的进行数据预设或存储

    不支持HTML5标准的浏览器中，可利用 jQuery 实现数据存取



jQuery提供的存储接口

    jQuery.data( element, key, value )   //静态接口,存数据

    jQuery.data( element, key )          //静态接口,取数据   

    .data( key, value )                  //实例接口,存数据

    .data( key )                         //实例接口,取数据    

    2个方法在使用上存取都是通一个接口，传递元素，键值数据。在jQuery的官方文档中，建议用.data()方法来代替。



2个对应的删除接口

    jQuery.removeData( element [, name ] )

    .removeData( [name ] )

$('.left').click(function() {
    var ele = $(this);
    //通过$.data方式设置数据
    $.data(ele, "a", "data test")
    $.data(ele, "b", {
        name : "慕课网"
    })
    //通过$.data方式取出数据
    var reset = $.data(ele, "a") + "</br>" + $.data(ele, "b").name
    ele.find('span').append(reset)
})
