Mysql 用户管理

    登录 mysql 后

        mysql > set passwd=passwd('root');



如何对 MySQL 用户权限进行限制

    1. 创建新用户

        create user xiaoming identified by 'm123'


    2. 在默认情况下，新创建的 MySQL 用户，没有权限


    3. 如何给用户分配权限

        grant all on *.* to 'xiaoming'@'%'

        这时小明就拥有了所有权限


    4. 看看如何更精准的控制用户的权限

        我希望给 xiaoming 的权限就是对 temp1 数据库的 test2 表进行查询

        grant select on temp1.test2 to 'xiaoming'@'%'


    5. 如何回收权限[一般说也是 root 操作]

        revoke 权限 on 数据库名.表名 from '用户名'@'host名'

        revoke all on *.* from 'xiaoming'@'%'
