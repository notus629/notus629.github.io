基本输出 - TP3.1


    a、通过 echo 等 PHP 原生的输出方式在页面中输出


    b、通过 display 方法输出

        // 在 Index 模块下的index 方法下调用
        // 会输出模板目录(Tpl)下的，Index 目录下的 index.html 模板
        $this->display(); 


    c. 分配变量可以使用 assign 方法

        // Index 模块下的 index 方法中
        $this->assign('hi', 'notus123');
        // 模板 index.html 中，默认定界符为 {}
        {$hi}
