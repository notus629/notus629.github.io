03. Notus 补充笔记



1. 显示页面，即显示模板 V 代表的 html 页面       

    在 Index 模块，index 方法中使用 $this->display()
       
    在 tpl 目录下新建 Index 目录（对应模块类），其下新建 index.html 文件（对应 index 方法)



2. 从 C 中将 data 传递给 V

    在 C 的方法中：

        $name="Leo" 
        $this->assign("data", $name);


    在 V 中使用

        {$data}来访问，{ } 为定界符


    修改 {$data} 的左右定界符

        'TMPL_L_DELIM' => '<{',      // 修改左定界符
        'TMPL_R_DELIM' => '}>',      // 修改右定界



3. 操纵数据库，从数据库中读数据

    $m = new Model('User');
    $arr = $m->select();
    var_dump($arr);



4. 操作 DB 之前，必须先进行数据库的配置

    进入 应用/Conf/config.php

    "DB_TYPE" => "mysql",        // 设置数据库类型
    "DB_HOST" => "localhost",    // 设置主机名
    "DB_NAME" => "thinkphp",     // 设置数据库名
    "DB_USER" => "root",         // 数据库的用户名
    "DB_PWD" => ""，             // 数据库的密码
    "DB_PORT" => "3306",         // 数据库的端口号，一般可省，一般默认端口号为3306
    "DB_PREFIX" => "tp_",        // 表前缀，建表的时候对表的表名给予一个前缀，防止库在表在多个项目中使用时的同名冲突
