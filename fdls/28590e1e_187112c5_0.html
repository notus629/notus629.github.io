jQuery对象转化成DOM对象

    它们都是可以操作的DOM元素

    jQuery是一个类数组对象，而DOM对象就是一个单独的DOM元素。



jQuery对象转成DOM对象

    a. 利用数组下标的方式读取到jQuery中的DOM对象

        举例：

1
2
3
<div>元素一</div>
<div>元素二</div>
<div>元素三</div>

        转换：

1
2
3
var $div = $('div') //jQuery对象
var div = $div[0] //转化成DOM对象
div.style.color = 'red' //操作dom对象的属性


    b. 通过jQuery自带的get()方法

1
2
3
var $div = $('div') //jQuery对象
var div = $div.get(0) //通过get方法，转化成DOM对象
div.style.color = 'red' //操作dom对象的属性
