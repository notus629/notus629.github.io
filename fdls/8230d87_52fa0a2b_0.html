CURD - 增删改查 - TP3.1

    使用模型的实例可以对数据进行操作，操作的工作一般就是对数据库进行 增删改查 CURD

        // 创建数据库实例, 'User' 为表名
        $m=M('User');
        $arr=$m->select();
  

    CURD     

        增 -C Create     $m->add()
        删 -D Delete     $m->delete()
        改 -U Update     $m->save()
        查 -R Read       $m->select()
