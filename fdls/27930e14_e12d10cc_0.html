获取和设置 html 特性

    .attr(), 读取或设置指定的属性

        更新一个不存在的属性，会创建此属性并赋值

        // 读取指定的属性
        $('li#one').attr('id');

        // 设置(更新)指定的属性
        $('li#one').attr('id', 'hot');


    .removeAttr(), 移除指定的属性（及其属性值）

        $('li#one').removeAttr('id');


    .addClass(), 向 class 属性已有的值中添加一个新的值


    .removeClass(), 从 class 属性中移除一个属性值（保留其他属性值)
