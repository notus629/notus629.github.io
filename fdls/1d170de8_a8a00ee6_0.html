Blade 

    是 Laravel 提供的既简单又强大的模板引擎

    Blade 并不限制在视图（view）中使用原生的 php 代码

    所有 Blade 视图页面都将被编译成原生的 PHP 代码并缓存起来，除非你的模板文件被修改了，否则不会重新编译



模板继承

    section

    yield

新建一个模板文件

    // 一般用来放置公共部分
    /resources/views/layout.blade.php


    @section 

        @section('sidebar')

        用来定义视图片段

        即可在页面布局中定义视图片段，也可以有内容，还可以被子模板扩展

        示例：
            <div class="header"> 
                @section('header')
                头部
                @show 
            </div>

            <div class="content">
                @yield('content', '主要内容区域');
            </div>


    @yield 

        @yield('content', '主要内容区域')

        用来展示某个指定section所带来的内容，可想象成占位符，用子模板去实现

        不可扩展，在布局中只定义声明了一个视图片段，没有任何内容

        例：
             <title>@yield('title')</title>



子模板中继承

    子模板中如何继承 layout 中的布局?

    // 若被继承的模板在 views 目录下(layouts.blade.php)
    @extends('layouts')

    // 重写头部
    @section('header')
        @parent
        header
    @stop

    // @yield 的使用
    @section('content')
        content
    @stop
