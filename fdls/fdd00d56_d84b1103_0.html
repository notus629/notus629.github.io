防止和其它库冲突



方法一： 在其他库之后引用 jQuery

    jQuery 获得了 $ 优先使用权

        <script src="other.js"></script>
        <script src="jquery.js"></script>
 


方法二： 在脚本的开始使用 .noConflict() 方法

    .noConflict(), 释放 $ 缩写，使其它脚本可以使用 $, 之后使用全称来使用 jQuery

    $ 用 jQuery 来代替

        jQuery.noConflict();



方法三： 把脚本放到一个立即执行函数(IIFE)中，在立即执行函数中使用 $

    jQuery.noConflict();
    (function($) {
        $('div').hide();
    })(jQuery);



方法四： 自定义一个别名，如 $j

    var $j = jQuery.noConflict();
    $j(document).ready(function(){
        $j('div').hide();
    });



方法五： 向 .ready() 方法调用的匿名方法中传入 $ 作为参数

    jQuery(document).ready(function($){
        $('div').hide();
    });

    // 或
    jQuery(function($){
        $('div').hide();
    });
