需求 3

    在添加商品的时候，我们希望多两个信息，就是付费方式，还希望多一个邮资信息，我们又该怎样处理？
 
    引出修改内容模型的问题。



步骤如下

    1. 核心 -> 频道模型 -> 内容模型管理 -> 选择 shop 模型 -> 更改

    2. 添加新字段

    3. 如果希望看到效果，根据页面静态化原理，应该去修改内容页对应的模板，也需要对模板进行修改。首先要确定模板文件的位置（栏目管理->修改栏目->高级选项）

    4. 对内容模板修改, 增加如下代码
        <small>支付方式：</small><span>{dede:field.paymode/}</span>
        <small>邮资信息：</small><span>{dede:field.postmode/}</span>

    5. 然后重新生成内容页面即可



思考题： 

    1. 栏目（类别）是保存在哪个表中的？

    2. 一篇文章，一个商品，它们最终是存放在数据库的哪些表中？

    答：栏目是保存在 dede_arctype表中，一篇文章，一个商品是保存在 dede_archives 和对应的附加表 dede_addonxxx 表中。
