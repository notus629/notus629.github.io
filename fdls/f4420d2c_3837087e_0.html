安装步骤

    1. 安装centos7

    2. 配置网络

    3. 安装wget

    4. root身份安装

        a. $ wget http://www.we7.cc/download/lnmp/lanmp_wdcp_v3.tar.gz        ## 下载安装包lanmp + wdcp

        b. $ tar zxvf lanmp_wdcp_v3.tar.gz                                    ## 解压到当前目录

        c. $ lanmp.sh                                                         ## 直接安装脚本

        d. 安装的4个选项，nginx, apache, nginx+apache, all               
          
            选2， 即 nginx + php + mysql + zend + cAccelerator + pureftpd + phpmyadmin

    5. 安装完成后，登录WDCP后台，浏览器中打开 ip:8080, 在站点管理里创建一个站点

        a. 微擎默认帐号：admin 密码：we7.cc  WDCP后台默认账号：admin 密码：wdlinux.cn。

        b. 根据提示，修改WDCP登陆密码和mysql密码        

        c. 绑定相应的域名，指定目录到/www/web/we7 （域名若没有，随便指定一个不存在的）（实际这里指定的是/www/web/we7/public_html;而这个根目录并未被作为nginx的web根目录，因为被其它配置所覆盖）        

        d. 解决被覆盖的问题后，再将目录改为 /www/web/we7
    
    6. 相关设置，参见[Ref 2]


安装问题

    1. cenos7安装卡顿

        已解决。采用4核CPU，3G内存，最小化安装（无图形界面）

    2. 安装问题。

        已解决。步骤5b中，指定目录并非实际的web根目录，因为nginx存在错误，没有www用户和www组，导致这一步根目录设置示生效，但WDCP并未给出提示。

        解决方式。正确配置nginx之后，删掉conf/vhost/00000.default.conf。
