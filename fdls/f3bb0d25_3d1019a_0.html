Weex

    A framework for building Mobile cross-platform UI.

    Support Android 4.1 (API 16) and iOS 8.0+.

    Weex put Vue 2.x as its built-in JS Framework, and Vue has already been able to develop native mobile app.

    阿里无线前端抛出的 无线电商动态化解决方案 Weex.

    Weex 是一款轻量级的移动端跨平台动态性技术解决方案。

    叫 Vue-Native 也不为过



Weex Playground

    用于展示 Weex 新功能和炫酷特效的移动应用，它同时提供开发者通过扫描二维码的方式进行在线调试。
