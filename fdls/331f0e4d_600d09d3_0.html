模板中生成 URL 的 3 种方式

    url() 

        通过路由的名称生成 URL


    action()

        通过指定控制器及方法名来生成 URL


    route()

        通过 route 的别名来生成 URL



示例

    // url 为路由的名称
    <a href="{{ url('url') }}">url()</a>

    // StudentController 的 urlTest 方法
    <a href="{{ action('StudentController@urlTest') }}">action()</a>

    // 别名也为 url
    <a href="{{ route('url') }}">route()</a>
