PHP 开发框架

    laravel, 视频：laravelcast，Laravist [2]

    CodeIgniter: CI

    ThinkPHP: TP（国产），相对于laravel来说，CI和TP都属轻量级。

    Smarty: 模板引擎，而TP是MVC全能。如果非要选就选TP吧。但是建议先学smarty简单点。

    Symfony2: Drupal采用

    Zendframework2: magento使用

    Yii2: 国内很多公司使用

    Blade: 模板引擎，Laravel 采用的模板引擎

    Workmen: Workerman 是一款纯 PHP 开发的开源高性能的 PHP socket 服务器框架。被广泛的用于手机 app、移动通讯，微信小程序，手游服务端、网络游戏、PHP 聊天室、硬件通讯、智能家居、车联网、物联网等领域的开发。 支持 TCP 长连接，支持 Websocket、HTTP 等协议，支持自定义协议。拥有异步 Mysql、异步 Redis、异步 Http、异步消息队列等众多高性能组件。


        
TP 与 Smarty

    Smarty是一个模板引擎，主要处理 MVC 中的 V；

    Thinkphp是一个实现了 MVC 的框架，其中 V 的部分可以用 Smarty 驱动，默认是用它自己的引擎。

    网友建议：Smarty没必要去学了，一个原因是tp本来就有一套模板引擎，你会了tp就差不多等于回了smarty，另外一个原因是smarty用的越来越少了，火了一段时间之后，大家都慢慢倾向于原生的php输出
