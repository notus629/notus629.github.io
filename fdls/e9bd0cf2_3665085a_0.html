节点操作 - 即在选择器选中的元素上执行操作

    约定： . 开头的表示 jQuery 方法

    jQuery 对象 length 属性：描述这个 jQuery 对象中元素的个数。


    内容操作： 获取或更改元素的内容、属性以及文本节点。

    查找元素： 查找并选择元素，以便对 DOM 进行操作和遍历

    尺寸/位置： 获取或更新元素盒模型的尺寸或位置。

    特效和动画： 向页面的部分内容添加特效和动画

    事件： 为选中内容的每个元素创建事件监听器



链式操作

    在同一个选取结果上可以使用多个 jQuery 方法，同时列出这些方法，并用点号隔开，即链式操作  

    两种排版, 举例：

        $('li[id!="one"]').hide().delay(500).fadeIn(1400);

        $('li[id!="one"]')
            .hide()
            .delay(500)
            .fadeIn(1400);


    注：

        1. 多数来更改 jQuery 选取结果的方法都可以进行链式操作，但用来从 DOM 或浏览器中获取信息的方法则不能进行链式操作。  

        2. 若链式操作中的一个方法不能工作，则剩下的那些也不会再运行了。



循环

    默认为隐式迭代，即 jQuery 选取结果可以更新所有元素

    若需要从一组元素获取信息时，使用 .each 方法，而不是编写循环
