变量调节器

    首字母大写capitalize

        {$articleTitle|capitalize}

            
    字符串连接cat

        {$articleTitle|cat:"yesterday."}

        {$articleTitle|cat:"yesterday.":"asdfafd"}

            
    日期格式化 date_format, $yesterday为unix时间戳格式

        {$yesterday|date_format}             // Mar. 3, 2016

        {$yesterday|date_format:"%H:%M:%S"}  // 18:50:34 跟时区相关，默认greenswitch

        {$yesterday|date_format:"%B:%e:%Y"}  // 月，日，年

        
    为未赋值或为空的变量指定默认值 default

        {$artileTitle|default:"no title"}

            
    转码 escape，html转码，url转码，
        
        // url转码, 因为某些符号会影响脚本语言的正常运转，避免影响后续处理
        {$url|escape:"url"}
  
              
    小写lower, 大写upper

        {$articleTitle|lower}

        {$articleTitle|upper}
  
      
    nl2br, 换行符替换成<br />, 同php中的nl2br()函数

        {$articlecontent|nl2br}

            

其他的函数（参见手册）

    原则上应该通过php直接处理完再赋值到smarty变量里，少用smarty函数
