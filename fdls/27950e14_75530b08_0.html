vue 与 angular 区别

    vue - 简单、易学

        指令 v-XXX

        一片 html 代码配合上 json, 在 new 出来的 Vue 实例

        vue.js 是由个人维护的项目

        适合：移动端项目，小巧


    angular - 上手难

        指令 ng-xxx

        所有属性和方法都挂到 $scope 身上

        angular 由 google 维护

        适合：pc 端项目



共同点

    不兼容低版本 IE 浏览器

    一切以数据为中心，只要数据更改，一切就都更改了
