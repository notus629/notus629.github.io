过滤器

    {{12}}  {{'welcome'}}           // 即，模板里可以放数字、字符串、vue变量

    过滤器：即过滤模板数据

    系统提供一些过滤器



格式

    {{msg | filterA}}

    {{msg | filterA | filterB}}       // 多个过滤器

    {{msg | filter 参数}}



示例
   
    {{'welcome'|uppercase}}           // 大写

    {{'welcome'|capitalize}}

    {{'welcOME'|lowercase}}

    {{12 | currency}}                // $12.00

    {{12 | currency '￥'}}           // ￥12.00
