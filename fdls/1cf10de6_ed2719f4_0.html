查询构造器

    query builder

    使用 PDO 参数绑定，以保护应用程序免于 SQL 注入，因此传入的参数不需额外转义特殊字符

    基本可以满足所有数据库操作，而且在所有支持的数据库系统上都可以执行



新増数据

    // 返回 bool 类型
    $bool = DB::table('student')->insert(
        ['name' => 'imooc', 'age' => 18]
    );

    // 插入后返回 ID
    $id = DB::table('student')->insertGetId(
        ['name' => 'sean', 'age' => 18]
    );

    // 插入多条数据
    $bool = DB::table('student')->insert([
        ['name' => 'name1', 'age' => 20], 
        ['name' => 'name2', 'age' => 21]
    ]);
