jQuery 方法 

    attr() 获取和设置属性元素值

        attr(传入属性名)：获取属性的值

        attr(属性名, 属性值)：设置属性的值

        attr(属性名,函数值)：设置属性的函数值？

        attr(attributes)：给指定元素设置多个属性值，即：{属性名一: “属性值一” , 属性名二: “属性值二” , … … }



Atrribute 和 Property

    都可以翻译成属性，可用 "特性" 与 "属性" 来区分


    Attribute 是 DOM 节点自带的属性

        如 html 中常用的 id, align, title 等

        使用 attr() 方法


    Property 是这个 DOM 元素作为对象，其附加的内容

        如 tagName, nodeName, nodeType

        使用 .prop() 方法进行取值或赋值等
