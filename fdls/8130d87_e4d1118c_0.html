页面布局

    Drupal 的页面布局方式是将页面划分成不同的区域，如页头、上部、中部左侧、中部右侧、下部、页脚等等。然后在不同的区域放置相应的区块。



区块

    Drupal 的区块是页面的基本单位，将区块放置在指定的区域中，用来展示静态或动态的内容。



静态内容

    静态的内容可以用手动添加区块的方式，如图片、图片+文字等。

    借助 ckeditor 编辑器等扩展模块，在手动编写模块式还可以用 HTML 源码的方式。

    可以将不同的 CSS 样式应用到不同的区块，使显示更加符合网站的风格。

    区块的样式也可以在网站主题中统一定制。



动态内容

    动态内容通常是一组列表。

    区块内容一般由 views 等扩展模块来定制。

    可以是摘要方式、也可以是图片加标题等信息列表的方式，还可以是一组幻灯片，或者是定制的表格、相册等。

    展示的内容都允许随意添加链接，链接到相应的具体内容展示页面。
