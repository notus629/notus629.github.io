过滤器

    即过滤模板数据

    系统提供一些过滤器



格式

    {{msg | filterA}}

    // 多个过滤器
    {{msg | filterA | filterB}}       

    {{msg | filter 参数}}



系统提供的过滤器 - 示例
   
    {{'welcome'|uppercase}}          // 大写

    {{'welcome'|capitalize}}         // 首字母大写

    {{'welcOME'|lowercase}}          // 小写

    {{12 | currency}}                // $12.00

    {{12 | currency '￥'}}           // ￥12.00
