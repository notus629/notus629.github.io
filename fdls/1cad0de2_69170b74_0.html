02. Notus 补充笔记



1. 模块 & 方法                    

   访问应用的 3 种等同的 URL：

        http://localhost/thinkphp

        http://localhost/thinkphp/index.php

        // Index 模块（C）下的 index 方法
        http://localhost/thinkphp/index.php/Index/index   


   Index 模块

       对应：Lib/Action/IndexAction.class.php 文件的 IndexAction 类


   index 方法

       对应 IndexAction.class.php 中的 IndexAction 类的 index 方法

       还可以对应其它方法，对应地址栏地址： http://localhost/thinkphp/index.php/Index/show，即在 IndexAction 类下的 show 方法


    传值： 

        通过 URL 地址写入，http://域名/项目名/入口文件/模块名/方法名/键1/值1/键2/值2, 如
            http://localhost/thinkphp/index.php/Index/show/name/Tony/age/18

        再在模块方法中通过 $_Get['name'] 形式使用。


   rewrite 模式，省略 index.php，即 http://localhost/thinkphp/Index/index

       需要使 apache 服务器支持 rewrite 模式
           a. 在 Apache 配置文件 httpd.conf 中，去掉 rewrite 行的注释，重启服务
           b. 在入口文件同目录下（项目目录下），创建 .htaccess 文件，里面写入 rewrite 规则。

           <IfModule mod_rewrite.c>
                RewriteEngine on
                RewriteCond %{REQUEST_FILENAME} !-d
                RewriteCond %{REQUEST_FILENAME} !-f
                RewriteRule ^(.*)$ index.php/$1 [QSA,PT,L]
           </IfModule>




2. 配置修改                       

    路径为：应用/Conf/config.php

    
    修改地址中键值分隔符：即

        http://localhost/thinkphp/index.php/Index-show-name-Tony-age-18
    
        <?php
            return array{
                // 修改 URL 分隔符
                "URL_PATHINFO_DEPR" =>"-", 
            };
        ?>


    修改后不能即时生效问题：删除缓存
