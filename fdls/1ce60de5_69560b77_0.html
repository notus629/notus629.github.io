05. Notus 补充笔记



1. find, 查找单条数据

    $m->find()      // 返回 id 为 1 的数据
    $m->find(2)     // 返回 id 为 2 的数据



2. 连贯操作

    两个 -> ： 连贯操作

    // 默认也是获取 id 为 1 的 'username' 字段的值
    $m->getField('username') 

    // 获取 id 为 2 的 'username' 字段的值
    $m->where('id=2')->getField('username')        

    // 或此处与直接 getField('username') 一样，where 没起作用
    $m->where(2)->getField('username') 



3. 没有让模板显示，则 trace 信息没有

    若注释 $->display()，则 trace 没有
